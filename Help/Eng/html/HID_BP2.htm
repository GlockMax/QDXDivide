Optional Stop

In the Main Menu: Work > Optional/Program Stop

Quick activating:  icon.

This command enables or disables the possibility to stop the simulation or to move the machine tool in the line that goes after the Optional Stop program code. Usually there are M0 and M1 codes. In fact it can be any of the codes defined in the CNC control file in the Section MiscFunctionDefinition as:

Stop            0

OptionalStop                   1

 

For example, Optional Stop is enabled ( icon is pushed), simulation mode is Smoothly (). Then if you push (To Breakpoint) button, the machine tool will move to the next line after the one with M0 or M1 code.

Edge compensation

This command calls the dialog to set offset values, insert radius and insert offset direction number.

Number - compensator number

X-offset - X offset relative to base point of insert.

Z-offset - Z offset relative to base point of insert.

Radius - rounding radius of insert edge.

Direction - cutting edge direction code.

Number + - add compensator to table. List of compensators used in current project is taken from CNC programs automatically.You may add additional compensator numbers by pressing "+".

Tool nose PT (Direction) - descriptive picture for correct direction code setting. Axes directions: blue for +Z axis, red for +X axis.

Multi edge tool setting

Let's consider the setting of multi edge tool.

There may be several points on a single insert, which take part in machining. The CNC program switch between these points using different compensators. For example, grooving tool. Right edge is set by direction 4 compensator 25 (T425), left edge - position 4 compensator 36 (T436).

Let's set the grooving tool 3 mm width for cutting outer groove. To do this, proceed to insert creation dialog, set the width T=3mm and select the left edge as a base point (Nose number 1). As an insert has R1=0.2, and the default point is in the center of this radius, you should shift the base point along X and Z axes in negative direction by 0,2 mm. Push "-Z"and"-X".

Let's set the right edge. To do this, let's make an offset along Z axis relatively to previously created base point (left edge). This offset is made by calling "Edge compensation" in compensator number 25. Set Z-shift size equal to insert width, 3mm. For the left edge we use compensator number 36. This compensator should be blank, as this edge is defined as base point.

Example of CNC-program using this tool.

CNC program:

T436 Tool change, right cutting edge
G0 Z-8. X22. approaching the right edge of a groove
G1 F.1 X15. cutting
G0 X22. exiting the groove
T425 changing to left cutting edge
G0 Z-12. approaching the left edge of a groove
G1 F.1 X15. cutting
T436 changing to right cutting edge
G1 F.2 Z-8. finishing groove bottom
X22. exiting the groove

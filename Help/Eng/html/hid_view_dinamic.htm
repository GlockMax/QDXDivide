"Highlight" Command (View Menu)

While moving the mouse over the graphic window, element pointed by the mouse for some period of time will be highlighted. Also you will see the hint about the interpolation type (G0, G1, G2, G3) and moving direction in selected line. At the same time all the geometric information about highlighted element (start point, end point, angles with coordinate axes, element length, arc center) is available in the Geometry tab  in the Control Panel. All this actions take much of computational resources and it can result in slowing down the work if computer is not powerful enough. To avoid this situation you can use  the "Highlight" command to turn on or off the elements highlighting. When you turn this function off the line number and CNC program name will not be shown in the status bar anymore.

Graphical window function

Graphical window is used to perform graphical operations on a project.

There are several graphical operations:

displaying the CNC program, tools, stock, part and machine;

Current view management (rotation, scaling and so on);

simulation display in one of the modes with material removal ( - Smoothly, - Quick , - Result) or without material removal ( - Wireframe) ??? old help???;

viewing geometrical parameters of the stock being processed;

measurement and section operations .

There are two graphical windows, stock and machine.

The basic properties of the graphical window are:

background color;

This parameter set the background color for the graphical window. This parameter can be set in Settings > Parameters on the Design tab. Select GraphicalWindowBackground in Element combo box and click on the Color button, then select the color from the color dialog.

program color ;
stock color;
part color;
tools color;

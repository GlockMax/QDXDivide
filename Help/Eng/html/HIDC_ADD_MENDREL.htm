Holder Creation

A holder is a part of a cutting tool fixturing (tool fasteners).

This element allows to control the contacting of those parts of the machine that are not involved in the processing of the material with a workpiece.

Geometrical parameters:

Diameter (D)

A numeric field where you should enter the value of the holder diameter.

In order to create a tool this field can not possibly be empty or negative.

Height (H)

A numeric field where you should enter the value of the holder height.

In order to create a tool this field can not possibly be empty or negative.

Lower Point Diameter (D1)

A numeric field where you should enter the value of the diameter at the bottom of the holder. It can not exceed the value of the Diameter (D).lf you set this value equal to (D) you will create a holder which geometrical parameters will be described by the cylinder. In order to create a holder with conical surface you should set

Height of Conical Part (h)

A numeric field where you should enter the value of the height of the conical part.

Such parameters as Name and Displacement have the same value as for a Simple Tool Creation.

You can create a holder for several already existing tools that lack holders. To do so select one or several tools in the Tools List and click Add (>). To delete tools click Delete (<-). However you can not delete the current compound tool from the resultant list.

Stock properties

Stock properties are managed in Properties pane. Properties vary depending on the type of project - mill, lathe, mill-turn.

Mill project properties

Lathe and mill-turn projects properties

 

To change the desired property, click on the corresponding value in the right column, then select the value from list or enter the value from keyboard. Modified parameter will be highlighted bold. On the bottom of the panels you will see the hint for selected property.

Visibility - controls the visibility of selected program. "Yes" - the stock is visible, "No" - the stock is hidden.

Quick stock visibility toggle:

click on the lamp icon in object panel for stock object

 

Edges - controls edges visibility for stock. "No" for invisible edges, "Yes" for visible.

Edges colour - controls stock edges color.

Edge thickness - controls edges thickness. Floating point value from 1.0 to 10.0.

Angle of smoothing - manages the angle of smoothing. If an angle between faces is less than this value, the angle would be smoothed. Minimal value is 0. Also this parameter affects the definition of edges.

Value of transparency - controls the degree of transparency for "transparency" mode.

Colour - controls stock initial color.

Paint - controls the way the stock is displayed during machining. "Polychromy" - surface color corresponds the color of tool, which formed this surface; "Homochromatism" - the stock is displayed with single defined color.

Rendering - controls the way the stock is displayed. Three modes available: "Shade", "Mirror", "Transparency".

Wireframe - controls display mode for currently selected model. "No" - model is displayed solid with selected color, "Yes" - model is displayed as wireframe with selected color.

Stock type - displays the stock type (mill, turn...).

 

In lathe and mill-turn projects there are stock properties, controlling the notch by two angles.

 

Corner initial - an angle of notch start in degrees. This angle is laid off from X axis. Positive direction is counterclockwise looking from positive direction of Z axis.

Corner increment - increment for initial corner in degrees. This angle is laid off initial angle. Positive direction is counterclockwise looking from positive direction of Z axis. For the stock during simulation to be without notch, the increment should be 360 degrees.

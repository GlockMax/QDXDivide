Execute > N Lines" Command (Work Menu)

Quick activating:  icon or F3 button.

This command is used to execute specified number of program lines starting from the current position. You can set the number of lines to execute in the dialog box next to the icon. You can stop the execution process using Stop or Reset commands. If one of the solid simulation modes is turned on then you will get machined stock model.

All Breakpoints are ignored in this command. The execution goes only for the set number of CNC program lines.

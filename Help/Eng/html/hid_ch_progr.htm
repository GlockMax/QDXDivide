Program

It is a file containing the NC program text.

You right-click and the contextual menu is activated which contains the following items.

Save

To save the program text in a file.

Text

To show the NC program text in the textbox.

Delete

To delete a program from the project.

Color

To change the color of a cnc-program toolpath on the screen.

Visible

To hide or to show a cnc-program toolpath on the screen. The icon changes its color to a dark one.

Active

To select a program for future work. The icon changes its color to green.

DXF Export

To export a toolpath to an external file of the DXF. format.

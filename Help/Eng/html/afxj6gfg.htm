Simple Tool Creation

The most commonly used types of tools which diameter is less than the tool height belong to this category. The end mill, the conical mill, the spherical mill and drills are among such tools.

If you want to create a tool which diameter is far bigger than its breadth, then create a disk tool. Click Other ... and you will activate the dialog for creating this tool.

In this dialog the tools are divided into the following main types:

§ Tool with champher;

§ Tool with fillet at the top of the cutting edge;

§ Conical tool;

§ Drills.

In order to set a tool it is necessary to define its type by selecting a corresponding icon and to set the necessary parameters:

Main Parameters:

§Tool diameter in mm (D);

§Tool height (H);

§Radius at the top of the tool (R);

§Cone angle of the tool cutting part (a);

§Name – a tool gets the name automatically from its own parameters. The parameters being changed, the tool name will change. If you set the name manually, it will not change automatically;

§Displacement – tool displacement in Z-direction.

Additional Parameters:

§Size of the cutting edge (h).

In this mode a tool area which can process the material and one that can not are highlighted. The size of these areas can be defined by the tool height. These areas are highlighted with different colors that can be changed by right-clicking and activating either the Compound Tool menu or the Simple Tool menu

If you want to change the color parameters set by default, on the Options menu select the Parameters item and then the Design tab. Select ToolActivePartByDefault or ToolInactivePartByDefault and then the necessary color in the window appeared in the Element combo-box.

In order to help to set tool parameters there are schematic pictures of these types of tools in the right part of the dialog.

Properties of STL model

Some properties for STL models are managed from Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

Visibility - controls visibility for currently selected model. "No" for invisible, "Yes" for visible. Visibility of model depends on visibility of parent object in Object pane (Stock or Fixture).

Edges - controls edges visibility for currently selected model. "No" for invisible edges, "Yes" for visible. Edges visibility of model depends on edges visibility of parent object in Object pane (Stock or Fixture).

Colour - controls model color.

File - displays the path and file name for STL file.

Rendering - controls the way model is displayed. Three modes available: "Shade", "Mirror", "Transparency".

Wireframe - controls display mode for currently selected model. "No" - model is displayed solid with selected color, "Yes" - model is displayed as wireframe with selected color.

Smoothing - controls the smoothing of the model. "No" for no smoothing, "Yes" for smoothing.

"Create" Function (File Menu)

In order not to repeat all actions connected with creating CNC control, tools, stock and other parameters you can use a Project concept to see how your program works later. Project file (.ncm) contains all information about loaded NC programs, tool, stock and other data.

This File Menu function is used to create a new NCmanager project. In NCManager we have three types of projects. Milling, Turning, and Milling-Turning. This function creates project of active type. When NCManager creates new project, CNC control and machine tool will remain the same while CNC programs, stock and tool will be deleted.

Using Hot Key: CTRL + N

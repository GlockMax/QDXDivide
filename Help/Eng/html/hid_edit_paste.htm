"Paste" Function (Edit Menu)

This is one of the text window edit functions.

Use this function to paste current clipboard contents that has been copied there with Cut or Copy function.

You can also move selected part of the text with the help of mouse cursor. Press "Ctrl" while moving and your selected part of the text will be copied to the new place.

Other ways of activating this function:

Using Hot Key: CTRL + V

From the Text Window Context Menu

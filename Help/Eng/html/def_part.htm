Part

Load a part.

You right-click and the contextual menu is activated which contains the following items.

Load - From STL file

Quick activating:

This command allows to load a file of STL-format that describes a stock geometrical parameters. At that the dialog box for opening a document will be activated. There .stl filename extension will be set.

The icon next to "Part" will change when the part is loaded into the project.

Delete

To delete a stock from the project.

Color ...

Quick activating:

To change the color with the help of which a workpiece is displayed on the screen.

Visible

Quick activating:

:

To hide or to show a part in the course of processing.

Wire-frame

Show the stock in the wire-frame mode.

Transparency

To set the degree of transparency of a part.

N-hedron

Set N-hedron stock sizes.

Offset 

Change the diameter and height of the stock on the offset value if bounding box is pushed.

Bounding Box 

Stock is automatically set by sizes of work movement toolpaths of all visible project programs. Fast movements and hidden project programs are not taken into account. 

Sides Number N

Number of N-hedron sides. If it is "0", then cylinder will be drawn.

Angle A

This angle is used to set the orientation on the XY-plane.

Length L

Stock length.

External D1

External diameter from the left side of the stock.

External D2

External diameter from the right side of the stock.

Internal D1

Internal diameter from the left side of the stock.

Internal D1

Internal diameter from the right side of the stock.

Delta Z

Stock moving on the Z-axis. You can change the stock position comparing to the program "zero-point".

If D1=D2 then cylinder will be drawn. If D2= D4 then internal hole will be straight. If D3=0 and D4=0 then there will be no internal hole in this stock.

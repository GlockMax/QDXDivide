From STL File

To set the dimensions of a round workpiece.

Find

This command allows to load a file of STL-format that describes a workpiece geometrical parameters. At that the dialog box for opening a document will be activated. There .stl filename extension will be set.

In order to open a document either on the hard or net disk do the following:

1. Click Open or select the Open item on the File menu.

2. From the Folder list select disk, catalog (folder) or Internet catalog containing the document.

3. Double-click and open folders one after another until you open a catalog containing the desired document.

If you failed to find the file using this method then use special Windows features for searching files.

4. Double-click the desired document.

The opened file name will be indicated in the File field.

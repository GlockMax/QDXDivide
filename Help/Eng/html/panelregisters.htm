Registers pane

Registers pane displays the value of registers used in CNC program.

You can switch panel visibility in Settings > Panes.

You can change pane position on screen with a mouse, fit pane to any other pane, make it autohide.

You can set background color, size and color of text of Geometry pane. This parameters are set in Settings > Parameters... , Design tab. Select Element ФонПанели или ТекстПанели.

If register value is null, such register is not displayed.

If in current program line the register have changed its value, it will be highlighted red. If register value hasn't changed and in not null, it will be black.

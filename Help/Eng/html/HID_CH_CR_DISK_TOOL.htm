Complex Tool Creation

The so called disk mills which diameter is much bigger than breadth belong to this category.

To create a Simple Tool click Cancel to cancel this dialog.

In this dialog the tools are divided into the following main categories:

§ Disk mills with champhers;

§ Disk mills with fillets of the cutting edge;

§ Single-angle mills

§ Double-angle mills;

§ Tools described by APT 7;

§ Complex disk mills.

In order to set a tool it is necessary to define its type by selecting a corresponding icon and to set the necessary parameters:

Main Parameters:

§Tool diameter in mm;

§Tool breadth;

§Radii at the top of the tool and on the sides;

§Cone angle of the tool cutting part or the champhers;

§Name – a tool gets the name automatically from its own parameters. The parameters being changed, the tool name will change. If you set the name manually, it will not change automatically;

Additional Parameters:

§Size of the cutting edges

In this mode a tool area which can process the material and one that can not are highlighted. The size of these areas can be defined by the tool height. These areas are highlighted with different colors that can be changed by right-clicking and activating either the Compound Tool menu or the Simple Tool menu.

In order to help you set tool parameters in the right part of the dialog there is schematic pictures of these tool types.

Elementary Tool Menu

Elementary tool constitutes a compound one.

You right-click and the contextual menu is activated containing the following items.

Change

To set other parameters of a simple tool.

Cut

To delete the current selected tool from a compound tool and place it into the clipboard.

This command being executed, the data previously saved in the clipboard are replaced.

Copy

To copy the current selected tool into the clipboard.

This command being executed, the data previously saved in the clipboard are replaced.

Paste

To paste a constituent tool copied with the help of Cut or Copy from the clipboard into the current compound tool.

Delete

To delete a constituent tool from a compound one.

Color

To change the color with the help of which a tool is displayed on the screen.

Parameters of one elementary tool can differ from parameters of another one.

Active Part

To change the color of that part of the tool which is involved in processing

Inactive Part

To change the color of that part of the tool which is not involved in processing

If you want to change the color parameters set by default, on the Options menu select the Parameters item and then the Design tab. Select ToolActivePartByDefault or ToolInactivePartByDefault and then the necessary color in the window appeared in the Element combo-box.

Wire-frame

In the shade mode a tool is displayed as a wire-frame.

Parameters of one elementary tool can differ from parameters of another one.

Transparency

To set the degree of transparency of a tool in the shade mode.

Parameters of one elementary tool can differ from parameters of another one.

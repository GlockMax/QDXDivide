"View" Commands

One of graphic windows must be active. there are two graphic windows: Machine Tool and Stock.

For the quickness all View functions are brought out into separate bar. You can call it by pressing right mouse button on the graphic window.

Some of functions you can find in View toolbar:

Also you can control dynamic rotation, moving, scaling by pressing CTRL, ALT or SHIFT together with left mouse button.

Regenerate

recalculate toolpath image.
Highlight

turn on/off dynamic bindings while working with graphical window.
Zoom Window

enlarge the graphical window area.
Zoom

dynamically zoom out or zoom in the image.
Pan

move the image.
Zoom All

show on the screen all the contents of the graphical window.
Rotate XYZ

dynamic rotation around a random axis.
Rotate X

dynamic rotation around the X.

Rotate Y

dynamic rotation around the Y.
Rotate Z

dynamic rotation around the Z.
Redraw

redraw the image.
Save Pictures

save the view under a name.
Load Pictures

call the view saved earlier.
Standard

call one of the views. The whole image will be displayed.
Last view

go back to the previous view.

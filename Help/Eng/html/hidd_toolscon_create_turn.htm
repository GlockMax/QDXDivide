Creating lathe tool

This command is used to set tool type, form and dimensions. There are two types of tools - lathe tool and milling tools. Lathe tool can be created only in lathe or mill turn projects.

As well as milling tools, lathe tool can be complex.

Lets take a look at lathe tool creation. When you click on Create…, following dialog appears:

Tool parameters

Name

Tool name. It is formed automatically and is shown in tool tree after its number. You may set the name manually.

Cutter ID

Tool number. Tool is loaded by program according to this number.

Axis

Tool axis number. This is a special concept, used to set the tool at any angle to the spindle axis. Tool orientation and axis number are set in machine description file.

Turret

Tool bank number.

Marker

Defines the size of a tool marker. Then the tool is hidden or not defined, it is displayed by its marker. Default marker size is set in Settings>Parameters...>Graphics Tool Marker.

X Shift

Tool shift along X axis.

Z Shift

Tool shift along Z axis.

Rotation

Tool angle in XZ plane. This parameter specifies complex tool orientation, in simple case - insert and holder.

Tool elements

Any lathe tool can consist of a set of inserts of any type and any number of tool holders. We will call this tool complex. This list displays the names of tools, which form a component tool. Double-click on an element name to call its edit dialog.

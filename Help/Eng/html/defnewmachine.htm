New Machine

You can find the cinematic definition files of the specific machine tool in ..\def\machines\"Machine Name". "Machine Name" folder name will be shown in the list of machine tools when you load this machine into NCManager.

In the "Machine Name" folder you can find the files with driven elements models in STL format and XML file with the fixed name - "main.xml" with cinematic definition.

"main.xml" file

You have to define names and machine axes types in this file. Also you have to define their interaction and snap solid models in STL formats to the axes.

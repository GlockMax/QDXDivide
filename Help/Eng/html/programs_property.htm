Properties of CNC-Programs

CNC-Programs properties are managed in Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

Visibility - controls visibility for CNC-Programs toolpaths. "No" - all toolpaths are invisible, "Yes" each CNC-Program toolpath visibility is controlled by its own visibility parameter.

Toolpath - controls the type of toolpaths to be displayed in graphical windows. Three options available: "Ignore cutter compensation", "By cutter compensation", "Both". During machining process, tools center moves along toolpath with cutter compensation. Compensation value for each tool is set in SNS-Control>Start state...>Compens.

Ignore cutter compensation - only programmed toolpath without tool compensation is displayed (yellow toolpath on the picture)

 

By cutter compensation - toolpath with diameter compensation or turning compensation is displayed (blue toolpath on the picture).

Both- display both programmed and compensated toolpaths. Color of toolpath with compensation is an inverted color of programmed toolpath. (on the picture - yellow toolpath without compensation, blue - with compensation).

 

To show - controls the mode of drawing toolpaths. Three options available: "Full toolpath", "Worked-out toolpath", "Fragment".

Full toolpath - all toolpaths of all cnc-programs in project are displayed.

 

Worked-out toolpath - only the part from the start to current cnc-program line (excluding current line) is displayed. This mode makes the effect of wireframe simulation, i.e. step-by-step toolpath drawing according to cnc-program execution.

 

Fragment - only the part of toolpath corresponding to "Fragment start" and "Fragment end" parameters is displayed.

 

 

 

Paint - controls the way the toolpath is colored in graphical windows. Six options available: "By default ", "Interpolation type ", "Diameter compensation", "Length compensation", "Cycle type", "Cooling type".

Fragment start - controls the number of cnc-program lines visible before current line.

Fragment end - controls the number of cnc-program lines visible after current line.

Length - displays the summary length of all toolpaths millimeters.

Number of lines - displays the number of lines in all cnc-programs of the project.

Time - displays the summary time of execution of all cnc-programs of the project.

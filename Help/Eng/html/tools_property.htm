Tools properties

Some common tool parameters are managed from Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

There are two sections of parameters: all tools properties (cutting part and holder) and holder properties.

Properties for all tools:

Visibility - controls tool visibility when it's active (i.e. cnc-program passed the line for tool change). "No" for invisible, "Yes" for visible. When the tool is invisible, it's position is shown with a marker (square) colored like the cutting surface of the tool.

Quick tool visibility toggle:

Left click on tool icon (vertical rectangle).

 

Rendering - controls the way tool is displayed. Three modes available: "Shade", "Mirror", "Transparency".

Value of transparency - set the degree of transparency for"Transparency" mode.

Holder:

Visibility - controls tool holder visibility when it's active. "No" for invisible, "Yes" for visible.

Rendering - controls the way holder is displayed. Three modes available: "Shade", "Mirror", "Transparency".

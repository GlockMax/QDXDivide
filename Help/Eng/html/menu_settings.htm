"Settings" Commands

This commands are used for the interface and system working parameters adjustment. Here are such commands:

Design … – set of colors and types for the different interface elements

Control Panel … – define the Control panel elements.

Graphics … – set the graphic parameters.

Language - set the interface language. Russian or English.

Show Toolbars - define what icons do you want to turn on and their size.

Status  Bar - Show/Hide the Status Bar.

Axes - Show/Hide axes in the Graphic window.

Save … – save settings file for the next use. If you don't save the settings, you will have  the previous settings next time you run NCManager.

Collision detection

Menu item : Inspect>Collision...

Collision dialog allows to turn on and off collision detection and set parameters.

Components and tolerance

Define models which collisions to check. Set tolerance in numeric fields. If the distance between models is less than tolerance value, it is stated that collision has occurred.

Machine - toggle machine parts collision detection.

Tool - toggle collision detection with tool.

Part - toggle collision detection with part.

Fixture - toggle collision detection with fixture.

Collision options

Toggle collision detection and set additional parameters.

In background mode - detect collisions in background mode. In this mode collisions are detected even without simulation running. At the bottom of a main window an animated picture appears, indicating the process of collision detection.

While execution only - detect collisions during CNC-program execution. In this mode collisions are detected only when simulation is running.

Disable - disable collision detection.

Stop on collision - if checked, simulation would stop when collision occurs.

Collision detection step - accuracy in steps for collision detection in background mode. Minimal value is 1.

 

Collision detection is carried out only in Machine window.

There's an indicator at the bottom of main window, showing collision detection status. If it's gray - collision detection is off.

If the indicator is green - collision detection is on and no collisions occurred. If the indicator is red - collision detection is on and some collisions have occurred. Collision occurrences are listed in Problems pane.

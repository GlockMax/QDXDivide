CNC-Program loading

Load DEMO.NC program using one of the following ways:

1. Choose CNC-Programs in the Object Panel. Push right mouse button and press Load.

2. Or you can push Open icon on the left side of CNC-Programs.

Then in the WORK folder find DEMO.NC file.

Now the CNC Program is loaded. Let's learn how to operate with views in order to see DEMO.NC program toolpath.

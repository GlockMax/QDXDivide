"Save Pictures" Command (View Menu)

This command is used to save current view under the "1", "2", "3", "4" or "5" name.

First set the view you are going to save in the Graphic window and then save it under one of the names using the "Save Pictures" Command.

If there is a view already saved by this name, it will be deleted.

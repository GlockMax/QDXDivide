New Lines Numeration

This Edit Menu function is used to hide or renumber lines in the program text.

Set parameters and press "OK". Program text line numbers will be changed.

Press "Cancel" to exit this function.

Register - put any letter symbol of the line. If you don't need it - leave it blank.

Start - put the number to start numeration from.

Increment - put the number to increase next line's number to.

Maximum - put the maximum line number. After this number numeration will continue from the Start number.

Number Character - put the integer number that will mean number of digits in line's number. If the number of digits of the line's number is less then selected, necessary "0" digits will be added to the beginning of the line's number. This textbox is active if the Leading Zero checkbox is checked. For example. Number character is "4", Register is "N" and the Leading Zero checkbox is checked. Thus 5, 15, 165 and 1245 will be N0005, N0015, N0165 and N1245 correspondingly. If the Leading Zero checkbox is not checked then the line numbers will be: N5, N15, N165 and N1245.

Leading Zero - if this checkbox is checked you can set the Number Character value.

Insert Spaces - if this checkbox is checked then program will paste space symbol after Register value.

Delete Line Numbers - if this checkbox is checked then after pressing "OK" line numeration will be turned off.

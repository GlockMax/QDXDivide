Fixture

The Fixture element of Objects panel contains the STL models used as a fixture.

A project may contain any number of STL models.The names of loaded models are placed in Fixture list. Left mouse button activates Fixture context menu.

To expand or collapse the list double click the Fixture or press "+" or "-" to the left of it.

Pressing the right mouse button on Fixture element activates the context menu shown on a picture below.

Load > From STL file - loads an STL model into the project, activates a standard open dialog.

Fast call :

press on an icon on the Objects panel.

If any model loaded, the red sign near "Fixture" disappears. Also, additional submenu icons would appear, they control the visibility if STL models.

 

Any STL model is included in the project with its own file name, so it;s possible to load any number of files. It's possible to load the same file many times, in this case the file name is added with a number.Each model is assigned its own color from a color set.

Warning: the project contains only the link to STL file and its model position.

Delete all - deletes all models from fixture section.

Position - relocation, rotation and scaling for any (for all) models in project.

Unblank all - makes all models visible. The icons become yellow.

Blank all - makes all models invisible. The icon becomes gray.

Toggle Display - makes visible objects invisible and vice versa.

Properties - shows the Properties panel, to view or set some fixture parameters.

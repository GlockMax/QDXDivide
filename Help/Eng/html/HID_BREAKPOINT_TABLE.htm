Breakpoints

In the Main Menu: Work > Breakpoints

Quick activating:  icon.

This command will show you the Breakpoints dialog box

You can adjust Breakpoints by turning on or off the check boxes next to the Breakpoint name. Turning on the Use Additional Breakpoints button will let you stop the simulation process in the positions defined by the Additional Breakpoints. To quickly turn on or off all the Additional Breakpoints use this icon -  in the Breakpoints toolbar.

For example, on the Breakpoints dialog box picture above the simulation process will be stopped in such places:

  • if the feed value will be changed;

  • if there is only Z-axis move (generally in the penetration or machine tool pulling-out lines);

  •  lines where the diameter compensation was turned on.

What is the Breakpoint?

The simulation process always starts from the current machine tool position in the project CNC program text. We need Breakpoints to have an opportunity to set end points for the simulation or to find key positions of the program that we work with. Thus if some of the Breakpoints are turned on and we run To Breakpoint execution then the simulation process will be executed from current position to the first enabled Breakpoint. This position will become the new current position of the machine tool. The same result will be when running  (Forward) and  (Backward) commands. The machine tool will be moved to the new position with no simulation. This new position will also become current machine tool position in the CNC project's program.

Breakpoints are divided into two groups. Main and Additional. The only difference between them is that you can find Main Breakpoints in the Breakpoints toolbar also to control the process more quickly:

Here you can find four Breakpoints. They will stop the simulation process in the beginning or in the end of every CNC program, after the Breakpoint line, after machine tool changing line, or in the bookmarked line. 

Control of the Additional Breakpoints status can be held by pushing  (Additional Breakpoints) button. If you want to adjust the list of Additional Breakpoints then push  Breakpoints button.

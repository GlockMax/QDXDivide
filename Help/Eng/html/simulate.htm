Work Simulation

There are four modes of simulation. You can switch them from Work menu item or from the toolbox:

1. Result . In this mode you get machined stock without any simulation process.

2. Smoothly . In this mode you can see simulation process of cutting showing intermediate positions of each G-code line. You can modify simulation speed using a slider that appears when you choose this mode . You have an additional opportunity of moving simulation process forward and backward up to given number of lines while Pause button is pressed .

3. Quickly . In this mode you can see the simulation process of stock cutting with no intermediate positions of each G-code line. You can modify simulation speed using a slider that appears when you choose this mode.

4. Wireframe . In this mode there is no simulation process at all. Current stock will become hidden when you change simulation mode to Wireframe. You will see initial stock on your screen.

Choose one of a standard isometric views. Turn on the Machine window by selecting Both vertically in Window menu item. Hide the toolpath by disabling CNC-Program checkbox.

If, for example, we simply want to see Program running in Smoothly mode we have to select Smoothly mode  and press Reset  button in order to proceed to the beginning of the program. Now press To breakpoint icon . NCmanager will now start simulation process of deleting stock material. To stop the simulation push Stop button. After the end of simulation you will see the following:

Examine the finally machined stock by changing the view and enlarging parts that you are interested in. Also you can make sections  or measure distance and hole diameters in the Stock window by using Measure item . Pay attention to the fact that when you are in Measure command  and you are simply moving mouse cursor on the machined stock - you will see the surface type and its parameters.

Now save your results in NCmanager project.

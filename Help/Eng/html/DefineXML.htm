Настройки в файле define.xml

Ряд настроек по умолчанию хранятся в файле default.xml. В файле можно настроить размеры у токарных пластинок, а у ISO пластинок дополнительно задать форму. Настроить диапазон движка изменения скорости имитации в режимах Каркас, Плавно Быстро. Настроить скорость имитации для токарной и фрезерной имитации. Файл должен находиться в корневой папке NCManager.

Токарные пластинки по умолчанию. <Tools> <Turn> <Blade>

Тэг <Slot>

Задать параметры для прорезной пластины.

<Slot A = "0" C = "5" Edge = "1">

A - Угол наклона средней режущей кромки.
C - Задний угол.
Edge – номер режущей кромки. «0» - левая. «1» - правая.

В зависимости от типа единиц измерений можно задать размеры в миллиметрах (mm) или дюймах (inch).

<mm R1 = "0.2" R2 = "0.2" T = "2" L = "20" H = "1.36"/>

<inch R1 = "0.005" R2 = "0.005" T = "0.1" L = "0.5" H = "0.05"/>

R1 – радиус на левом углу пластины
R2 – радиус на правом углу пластины
T – ширина пластины
L – длина пластины
H – Толщина пластины

Тэг <Round>

Задать параметры для круглой пластины.

<Round C = "7">

C - Задний угол.

В зависимости от типа единиц измерений можно задать размеры в миллиметрах (mm) или дюймах (inch).

<mm R = "4" H = "3.18"/>

<inch R = "0.25" H = "0.125"/>

R - Радиус пластины
H – Толщина пластины

Тэг <ISO InsertForm>

Задать параметры для пластины по стандарту ISO.

<ISO InsertForm = "R" C = "5">

InsertForm - Код формы пластины (квадрат, прямоугольник, ромб, и т.д.).
C - Задний угол.

В зависимости от типа единиц измерений можно задать размеры в миллиметрах (mm) или дюймах (inch).

<mm L1 = "12.0" L2 = "20.0" R = "0.4" H = "3.18"/>

<inch L1 = "0.65" L2 = "0.75" R = "0.0156" H = "0.125"/>

L1 - Длина главной режущей кромки
L2 - Длина вспомогательной режущей кромки
R - Радиус пластины
H – Толщина пластины

Тэг <General>

Задать параметры для произвольной пластины.

<General A = "5." B = "80." C = "5.">

A – Угол с осью X.
B – Угол в плане.
C – Задний угол.

В зависимости от типа единиц измерений можно задать размеры в миллиметрах (mm) или дюймах (inch).

<mm L1 = "12.0" L2 = "8.0" R = "0.5" H = "4.76"/>

<inch L1 = "0.61" L2 = "0.433" R = "0.0156" H = "0.125"/>

L1 - Длина главной режущей кромки
L2 - Длина вспомогательной режущей кромки
R - Радиус пластины
H – Толщина пластины

Настройка диапазона движка управления скоростью имитации

Тэг <Sliders>

Настраивается три величины. Скорость при крайних положениях движка и положение движка по умолчанию.

Режим Каркас – задержка после выполнения каждого кадра в миллисекундах.

<Wire MIN = "10." MAX = "1000." DEF = "100."/>

MIN – Задержка в крайнем правом положении.
MAX – Задержка в крайнем левом положении.
DEF – Задержка по умолчанию.

Режим Плавно – скорость в миллиметрах на один такт таймера (тэг TimerSTEP).

<Smooth MIN = "0.5" MAX = "50." DEF = "2."/>

MIN – Скорость в крайнем правом положении.
MAX – Скорость в крайнем левом положении.
DEF – Скорость по умолчанию.

Режим Быстро – скорость в миллисекундах. Результат имитации будет показываться через заданное число миллисекунд.

<Quick MIN = "1." MAX = "200." DEF = "80."/>

MIN – Скорость в крайнем правом положении.
MAX – Скорость в крайнем левом положении.
DEF – Скорость по умолчанию.

Настройка скорости имитации

Тэг <Speed>

Дополнительные настройки скорости имитации в режиме Плавно для токарной (Lathe) и фрезерной (Mill) имитации.

<Scale GEN = "0.5" LONG = "10." SHORT = "3." LENGTH = "49."/>

GEN – Коэффициент изменения скорости движения на рабочих подачах.
LONG - Коэффициент изменения скорости движения на длинных холостых ходах.
SHORT - Коэффициент изменения скорости движения на коротких холостых ходах.
LENGTH – Перемещение считается длинным, если оно больше этой величины.

<Timer STEP = “40”/>

TimerSTEP - Задается тактовая частота таймера имитации в миллисекундах.

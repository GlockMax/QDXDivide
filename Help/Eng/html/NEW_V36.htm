Новое в NCManager версии 3.6

 

Панель Объектов

Новый элемент в панели - Оснастка. Служит для размещения в NCManager моделей оснастки в формате STL.

Деталь. Можно загрузить несколько моделей STL в качестве детали.

Операции с STL моделями. Загрузить, Переместить, Повернуть, Масштабировать, Экспортировать в файл.

Токарная заготовка имеет три режима тонирования. Одноцветность, Многоцветность, Зеркальность.

Заготовка. Новый тип фрезерной заготовки, Контур вытянуть.

Контур может быть импортирован из DXF файла.

Текст каждой программы в отдельности может быть обнавлен (вернуться к оригиналу программы).

Новая Панель инструментов - Индикаторы.

Новая команда Главное меню -> Анализ -> Статистика программ. Выдается время обработки и длина пути инструментов.

Интерпретация

Поддержка специального файла проекта для запуска NCManager из любого внешнего приложения.

Поддержка большинства станочных циклов стойки Heidenhain iTNC-530.

Поддержка станочных циклов стойки Acromatik 840-D

Поддержка токарной стойки НЦ-31

Поддержка токарной стойки HAAS SL

Новые типы параметров для настройки коррекции на диаметр.

Возможность автоматической настройки коррекции на диаметр, равной радиусу инструмента.

Поддержка цилиндрической интерполяции.

Поддержка полярной интерполяции.

Инструмент

Новый тип фрезерного инструмента.

Токарный инструмент. Изменен диалог задания параметров у Произвольной и Резьбовой пластинок.

Имитация

Новый движок для регулирования скорости имитации.

Отображение и управление траекторией инструмента одинаково во всех режимах имитации.

В режимах имитации Быстро, Плавно, в текстовом окне показывается текущий кадр программы.

Можно настроить скорость имитации для различных режимов и типов обработки.

В режиме плавно можно настроить различную скорость для рабочих и холостых перемещений.

Можно настроить параметры токарных пластин по умолчанию,

Резание поворотом при полярной интерполяции.

Резание поворотом когда ось вращения заготовки параллельна оси вращения инструмента.(не реализовано до конца)

Резание поворотом при цилиндрической интерполяции.(не реализовано до конца)

Резание поворотом когда ось вращения инструмента перпендикулярна оси вращения инструмента. (не реализовано до конца)

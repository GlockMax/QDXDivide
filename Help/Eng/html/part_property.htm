Fixture/stock properties

Fixture and Stock objects properties are managed from Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

Visibility - controls visibility for Fixture/Stock object. "No" - all Fixture/Stock models are invisible, "Yes" each Fixture/Stock model visibility is controlled by its own visibility parameter.

Edges - controls visibility of Fixture/Stock edges. "No" - edges are invisible, "Yes" - edges visibility is controlled by own models visibility values.

Edges color - controls the color of edges.

Edges thickness - controls edges thickness. Floating point value from 1.0 to 10.0.

Rendering - controls the way model is displayed. If "Not defined", each model is managed by it's own parameter.

Value of transparency - controls the degree of transparency for "transparency" mode.

Smoothing - controls all Fixture/Stock models smoothing. "No" - smoothing is off. "Yes" - smoothing depends on each model own property.

Angle of smoothing - manages the angle of smoothing. If an angle between faces is less than this value, the angle would be smoothed. Minimal value is 0. Also this parameter affects the definition of edges.

Creating insert

Add insert

This command allows you to set type, dimensions and position of an insert.

Common insert parameters:

Name – tool name.

Offset X (DX) Offset Z (DX)

Offset parameters are used to set the setup point for the insert. By default it is the center of insert rounding. If the actual point differs from default, you can set this point by this two offsets.

+DX, +DZ, -DX, -DZ - fill in the offset amount and sign. Offset amount is equal to insert rounding radius.

0 - zeroing offset.

Nose-R compensation - fill in the compensators for nose-radius compensation.

C– insert rear angle.

H – insert thickness.

D – insert orientation angle.

This parameters allows setting the tool for inside of outside turning.

Insert's cutting edges are always highlighted by the color of tools working plane.

Arbitrary insert parameters:

A – main angle

T – auxiliary angle

L1 –main cutting edge length

L2 – auxiliary cutting edge length

R – rounding radius between cutting edges. Such inserts have two cutting edges and default programming point is at the center of rounding R.

Round insert parameters:

R – insert radius.

Insert cutting edge is highlighted, and default programming point is in the center of insert.

Cutoff insert parameters

T – insert width

L – insert length

R1 – left corner rounding radius

R2 – right corner rounding radius

A - middle cutting edge angle.

Nose number - define working insert corner.

Such inserts have three cutting edges and default programming point matches the point of intersection of left and middle cutting edges.

ISO standard insert parameters:

Insert form – insert form code (square, rectangle, diamond, etc.). Select the code from list.

L1 – cutting edge length or rounding radius for round insert.

L2 – second cutting edge length, for irregular polygon insert shapes.

R – rounding radius between cutting edges. You may select the value from the list or fill in you value.

Nose number - define working insert corner.

Such inserts have two cutting edges, except round inserts. Rectangular and parallelogram inserts have three cutting edges. Default programming point for all inserts matches the center of rounding radius R.

Thread-cutting insert parameters

A – thread angle

T – insert width

L – insert length

R – rounding radius between cutting edges.

K – distance between the edge and left corner of the insert.

Such inserts have two cutting edges, and default programming point matches the center of rounding radius R.

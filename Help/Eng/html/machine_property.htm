Machine properties

Machine parameters are managed from Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

Visibility - controls visibility of all machine acting mechanisms. "No" - all models are invisible, "Yes" visibility controlled by individual model visibility property.

Edges - controls visibility of all machine models edges. "No" - all edges are invisible, "Yes" visibility controlled by individual model edges visibility property.

Edge color - controls edges color.

Value of transparency - controls the degree of transparency for "transparency" mode.

Smoothing - controls all machine models smoothing. "No" - smoothing is off. "Yes" - smoothing depends on each model own property.

Angle of smoothing - manages the angle of smoothing. If an angle between faces is less than this value, the angle would be smoothed. Minimal value is 0. Also this parameter affects the definition of edges.

Tools - controls the visibility of virtual tools in the Machine graphical window.

Machine model properties

Machine model properties are managed from Properties pane.

Visibility - controls visibility for currently selected model. "No" for invisible, "Yes" for visible. Visibility of model depends on visibility of the machine.

Edges - controls visibility of all machine models edges. "No" - edges are invisible, "Yes" - edges are visible. Visibility of model edges depends on visibility of the machine edges.

Smoothing - controls all machine models smoothing. "No" - smoothing is off. "Yes" - smoothing is on. Smoothing of model depends on smoothing of the machine.

Rendering - controls the way model is displayed. Three modes available: "Shade", "Mirror", "Transparency". Rendering of model depends on Rendering property of the machine.

Wireframe - controls display mode for currently selected model. "No" - model is displayed solid with selected color, "Yes" - model is displayed as wireframe with selected color.

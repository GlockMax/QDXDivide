Starting NCManager

Run NCManager.

If this is the first time you run NCManager it will look like this:

Stock Window - in this window you can see the wireframe toolpath and the machine tool movement during the simulation process.

Machine Tool Window - in this window you can see the wireframe toolpath and the movement in accordance with kinematics machine tool scheme.

The visual difference between Stock Window and Machine Tool Widow is that you can see cube on the left window (conventional Stock window sign) around the co-ordinate sign but not on the right one.

Object Pane is designed for displaying and controlling the project main items.

Parameters Pane is designed for displaying of current state of CNC Control.

Editor Pane is designed for displaying and editing of CNC - program text.

ISO CNC Control is loaded by default.

CNC Control defines essential information in order to look at specific CNC machine tool program.

3x_DEFAULT machine tool is loaded by default.

3x_DEFAULT is a  vertical-milling machine 3X kinematics with no working body model. Machine tool table is moving in X and Y directions, mill spindle is moving in Z-direction.

The next step is CNC - program loading.

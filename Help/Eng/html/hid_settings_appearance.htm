Design Option

This dialog allows to set up different elements of the interface.

Main Parameters:

Element – a combo-box where you should select an element which options you would like to view or to change.

Color – the button shows the current color of the Element. This button being pressed, a dialog for setting up color is activated where you can select the color you like.

Font – this combo-box becomes available if the selected Element beside the Color parameters has other variable text parameters. Select the font type you like from the list of all possible fonts.

Size – as well as the Font field this one becomes available if the selected Element has text parameters that can be set up. You can select a value from the list or enter your own one.

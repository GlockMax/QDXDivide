Compound Tool Menu

A certain tool (a compound one) included in the project.

You right-click and the contextual menu containing the following items is activated.

Change

To set other parameters of the tool.

Save

To save tool parameters in an external file.

Cut

To delete the current selected compound tool with all the simple tools it contains from the list of tools and place it into the clipboard.

This command being executed, the data previously saved in the clipboard are replaced.

Copy

To copy the current selected compound tool with all the simple tools it contains into the clipboard.

This command being executed, the data previously saved in the clipboard are replaced.

Paste

To paste a compound tool copied with the help of Cut or Copy from the clipboard.

Delete

To delete the tool from the project.

Color

To change the color with the help of which a tool is displayed on the screen. A new value will be set for every tool which constitutes a compound one.

Active Part

To change the color of that part of the tool which is involved in processing.

Inactive Part

To change the color of that part of the tool which is not involved in processing.

If you want to change the color parameters set by default, on the Options menu select the Parameters... item and then the Design tab. Select ToolActivePartByDefault or ToolInactivePartByDefault and then the necessary color in the window appeared in the Element combo-box.

Visible

To hide or to show the tool in the course of processing.

Wire-Frame

In the shade mode a tool is displayed as a wire-frame.

A new value will be set for every tool which constitutes a compound one.

Transparency

To set the degree of transparency of a tool in the shade mode.

A new value will be set for every tool which constitutes a compound one.

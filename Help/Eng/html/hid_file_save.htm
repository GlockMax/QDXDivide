"Save" Function (File Menu)

This File Menu function is used to save active project with current name in current folder. When you are trying to save project without name - NCM will show you a dialog box of saving the project that will allow you to choose name for the project you are trying to save. If you need to change file name before saving - choose File > Save As.

Other ways of activating this function:

From Tool Bar:

Using Hot Key: CTRL + S

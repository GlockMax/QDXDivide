Program Start/Stop

In the Main Menu: Work > Program Start/End

Quick activating:  icon.

This command turns on or off the possibility to stop the simulation process or to move the machine tool in the beginning or in the end of the CNC project's program.

Let's suppose that Program Start/End command is switched on ( icon is pushed down), Smoothly () simulation mode is turned on, and there are more then one program loaded in the current NCManager project. When pushing  (To Breakpoint) icon the machine tool starts machining the stock. When the start or end line will be found, the simulation process will be stopped. To continue you have to push  icon again. Thus the simulation process will be stopped automatically before and after the end of the each program's simulation process.

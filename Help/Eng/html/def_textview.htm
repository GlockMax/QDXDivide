Editor

Editor is used to view and edit the CNC programs.

This window is in fact a simplified text editor with basic text editing functions, some special-purpose commands allowing to manage the program faster and easier, and problem-oriented commands for managing the program. Text editor commands are called from Edit menu.

You can adjust editor's appearance.

Font

NCM supports basic MS Windows fonts (Times New Roman, Arial, etc.). To change the editor's font open Settings > Parameters, Design tab. In the element menu select ProgramText, then select the font in the Font menu below.

Editor's background color;

To set editor's background color open Setting > Parameters, Design tab. Select TextWindowBackground in Element menu, then click on the Color field and select desired color in appeared color dialog.

Program text color

Each symbol of the CNC program refers to one of the groups. This behavior is defined in the CNC control definition file.

To change the symbol color use Settings > Parameters, Design tab. Select the appropriate Element in dropdown menu, then push the Color button and select desired color from the dialog.

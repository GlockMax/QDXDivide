Compound tool

A compound tool is a set of one or several elementary tools. As a result, a compound tool can form a non-convex contour. All its constituent simple tools are sorted out according to the displacement value as it is shown in the picture.

Maim Parameters:

Tool name. By default a tool automatically gets the name of its lowest constituent tool. If you replace the generated name by your own, then the automatic name generation mode will be unset. Remember that in this case a new name must not be “empty”.

Cutter ID. Tool position in the machine magazine;

Marker – the size in pixels of the “+” symbol which replaces a tool image on the screen when the tool itself is hidden.

Z Shift – tool shift in Z-direction;

Dialog Commands:

Add a simple tool;

Add a disk tool;

Add a holder;

Change – to change the parameters of a tool selected. To carry out this command you can double-click a tool name on the page.

Delete – to delete one or several tools selected.

More >> - allows to set additional tool parameters enlarging the dialog.

Additional Commands:

Cutting part – to set the color of the active part of the whole compound tool.

Not cutting part – to set the color of the inactive part of the whole compound tool.

Remember: these parameters will be set for al the tools that constitute the current compound tool..

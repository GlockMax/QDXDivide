Bookmark > Add/Delete (Edit Menu)

This Edit Menu function is used to create new bookmark on the current line. Added bookmark will appear as  icon.

To delete bookmark you need to set cursor on bookmarked line and press this button again.

If there are several bookmarks in the program text - you can move through them using Next Bookmark or Previous Bookmark.

Ways of activating this function:

From the Toolbar:

Using Hot Key: CTRL + V

From the Text Window Context Menu

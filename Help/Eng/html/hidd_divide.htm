Stock - Divide

Divide command is used to remove single piece of stock after machining simulation. You should select the desired way to remove pieces of stock in a dialog, and select desired pieces with mouse. To exit the command - close the dialog window.

Delete all except picked piece - the way to divide a stock, when only a piece picked by mouse will remain. As soon as a piece is picked, all other pieces of stock will be removed.

Delete only picked pieces - the way to delete only picked pieces. If there are several pieces you want to delete, you should pick them one by one in any order. As soon as a piece is picked, it will be removed. Removal process takes some time, but you can proceed picking pieces of while the process goes on.

Undo - revert stock to a state that was before command call.

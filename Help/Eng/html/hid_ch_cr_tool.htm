Milling tool creation

This command is used to set tool type, form and dimensions. There are two types of tools: milling tools and lathe tools. Milling tools can be set only in milling or mill-turn projects.

Lets state some definitions.

There are three types of elements, which form each tool. Standard tool, disk-shaped tool and holder. Any tool can be described as a set of any quantity of these elements. All the elements, which form a tool, are listed in tool tree in an offset descending order. On the following picture you can see tool number 1 named "Tool", which consists of three elements (10mm diameter drilling tool, 20mm diameter end-milling tool, and a holder).

In the simplest case tool consists of one element. On the following picture you can see tool number 2 named D90H100 (90 mm diameter and 100 mm high).

Example. To create a drilling tool define one element by calling Add standard tool. If you need the tool to have a holder, add one more element by calling Add Holder to Tool. As a result you'll have a tool, consisting of two elements, complex instrument.

Let's go through the milling tool creating procedure. After pushing Create... the following menu window appears:

Tool Parameters
Name

Name of the tool is formed automatically if not specified by user.

Cutter ID

Tool number. Tool is referenced from CNC-program by this number.

Axis

Tool axis number. This is a special concept, made to make it possible to create a tool with any angle to the spindle. Axis position and number are set in the machine description file.

Turret

Tool bank number.

Marker

Defines the size of tool marker. When a tool is hidden or a tool size and shape are not defined, tool's current position is marked by this marker. Default marker size os set in Settings>Parameters>Graphics>Tool Marker.

Z Shift

Defines tool's shift along the spindle axis.

Add standard Tool

Create end-milling cutters or drilling tools.

Add disk-shaped Tool

Create a special-type tools. Follow the notes od dialogs.

Add Holder to Tool

Create a holder to the current tool.

Tool elements

Any tool can be represented as a set of end-milling, disk-shaped tools and holders, i.e. complex tool. This list contains the names of tool's elements in Z shift descending order.

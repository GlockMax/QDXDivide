Graphics Option

This dialog allows to set up display parameters of the graphical window..

Variable parameters::

Net on the tool – you can set the following parameters of the net on the tool: High-Density, Middle, Low-Density. Selecting different values of this parameter you will set different density of the net with which the tools are displayed. High-Density net allows to display tools with the maximum quality of the image. However, different operations with tools (rotations, movement, scaling) will take more time than in case with a tool having a net with lower density (Middle, Low-Density values of the parameter). Consequently, if you select a Low-Density net operations with the tool will done the fastest possible however with some image quality losses. We recommend to work with the Middle tool net. It will guarantee an acceptable speed of operations at a sufficient level of image quality.

Circle approximation accuracy – this parameter sets the maximum deviation in pixels of a polygon that approximates the circle from the analytical circle.

Thus, the less the accuracy is, the “rougher” the circle of the driving toolpaths will be displayed (the fewer sides of a polygon are, the more the circle aliasing is.) We emphasize the fact that this parameter influences the quality of circles display only but not the accuracy of their view.

You can recalculate circles by selecting the View menu > the Regenerate item.

Selection area – the size of the square in pixels in which the elements are selected.

Tool Marker – the small rectangle's size which will show you the machine tool's position when the machine tool itself is hidden.

Snap Markers – this parameter is not used.

Size of coordinate axes 1

Size of coordinate axes 2

Size of coordinate axes 3 –sizes of the shown coordinate axes..

Enlargement Factor Loupe – this parameter is not used

Model Tolerance - the tolerance to draw the machined stock model.

Delay (mc) – a temporary delay at the step-by-step program execution.

Stock Multi-Colour On/Off - if it is checked and saved then in every next project the Stock Multi-Colour mode will be turned on. Otherwise it will be turned off.

Parameters pane

Parameters pane is used to display the geometrical, technological and other parameters, changing during program execution.The parameters are displayed for the current program line. Modified parameters are marked red.

You can show/hide Parameters pane from main menu Settings > Panes.

You can change the pane position on screen, dock it to any other pane, make it autohide.

Text Window Painting (Edit Menu)

This function performs thematic painting of the text window.

Such painting types are available:

Interpolation Type

There are three parts of toolpath and three types of paintings consequently:

Noncutting move;

Section Moving;

Arc Moving;

Tool Number

CNC program text parts are painted into the colors of corresponding machine tools.

Cutter Compensation Type

This way of painting is divided into two types

Length Compensation;

This type of painting depends on the toolpath direction:

  • Upward motion compensation;

  • Downward motion compensation;

  • Right motion compensation;

  • Left motion compensation;

Diameter Compensation;

This type of painting takes place when the Diameter Compensation is turned on.

Cooling Type

This type of painting is divided into such parameters:

No Cooling;

Air Cooling;

Liquid Cooling;

Feed Value

Red color is for the minimum Feed Value, blue color - for the maximum Feed Value, and green is for the middle Feed Value.

Cycle Type

This type of painting is divided into such parameters:

Drill cycle:

Thread cycle:

No Paint Mode

CNC program text will be painted using default painting rules defined in color settings.

To change the color of any program parameter except those indicated above, go to Settings > Parameters, Design tab. Then choose modifying Element and change its Color.

Ways of activating this function:

Control Panel:

From the Text Window Context Menu

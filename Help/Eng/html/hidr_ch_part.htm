Part model

File, containing part model in STL format.

Right click in a model name to activate context menu.

Commands of the context menu act on the selected part model. Part model selected in parts tree will be highlighted by its bounding box with a color of model.

Replace... - load another STL model in place of currently selected model. A standard file open dialog will appear.

Delete - remove selected part model from project.

Position... - shifting, rotation and scaling of selected part model.

Export... - save selected model to an STL file.

Properties... - show the "Properties" pane to view or set some part parameters.

"Go To > Forward" Command (Work Menu)

Quick activating:  icon.

This command is used to move from the current line forward. New position becomes current and is marked by the blue arrow in the text window. The machine tool is set to the new position in the graphic window. There is no simulation.

Program will move back to the previous Breakpoints. Turning on or off some of the Breakpoints you can move to the necessary CNC program parts (machine tool change, bookmark in the program text, and so on) that you have in the NCManager project.

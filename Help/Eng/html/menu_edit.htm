"Edit" Functions

Text window must be active - click left mouse button on it.

Most of "Edit" Functions are available from context menu that can be shown by clicking right mouse button on text window:

Some of "Edit" Functions you can find in the toolbar:

  Undo

Undoes last change in the text window.
  Redo

Redoes last undone change in the text window.
Cut

Cuts selected part from the text window and puts it in clipboard.
Copy

Copies selected part from the text window into clipboard.
Paste

Pastes text form the clipboard into the text window.
Select All

Selects all text from the text window.
  Find/Replace

Finds/Replaces text from the text window.
  Renumber Lines

Renumbers lines in the program text.
Insert Spaces

Inserts spaces after each word in the program text.
Delete Spaces

Deletes all spaces in the program text.
  Update

Updates all changes that has been made in the program text.
Line Numbers

Show/Hide text window lines numbers.
Bookmarks

  Add/Delete

  Next

  Previous

  Delete All
  Snap

On/Off possibility to move tool to the program line pointed in the text window.
  Paint

Shows text window painting types.

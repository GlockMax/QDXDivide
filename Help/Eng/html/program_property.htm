CNC-program properties

CNC-program properties in project are managed in Properties pane.

To change the desired property, click on the corresponding value in the right column, then select the value from list or enter the value from keyboard. Modified parameter will be highlighted bold. On the bottom of the panels you will see the hint for selected property.

Visibility - controls the visibility of selected program. "Yes" - the program is visible, "No" - the program is hidden.

Thickness - controls the thickness of toolpath of program, shown in graphical windows. The value is a floating point from 1 to 10.

Colour - controls the color of toolpath in the Objects pane, if the color mode is set to Default, for all programs.

Comment - set the comment for the program.

Length - shows the length of toolpath in millimeters.

Number of lines - shows the number of lines in program.

Time - shows the program execution time.

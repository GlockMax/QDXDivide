"Find/Replace" Function (Edit Menu)

This Edit Menu function is used to find necessary text in the program text.

Sample - textbox where you need to put search text. If you put search text in the textbox then Find Next and Select All buttons will be active.

Match whole word - program will search only whole mentioned word.

Match case - program will match case of the word you are looking for.

Regular expressions - you can use regular expressions in the search.

Find Next - every time you press this button program will continue searching program text before it finds next line with the text you are looking for. When search reaches the end of the text it starts searching from the beginning.

Select All - this button allows you to bookmark all program lines with the search text.

Other ways of activating this function:

From the Toolbar:

Using Hot Key: CTRL + F

From the Text Window Context Menu

 

Comment. To interrupt search - press ESC.

Contour

Create a stock by extruding the contour along Z axis for specified height. Contour is set in file with "StC" extension.

Stock context menu: Create > Extrude contour

Find - allows to load a file with contour geometry.

Length - displays X dimension of stock.

Width - displays the Y dimension of stock.

Height - displays the Z dimension of stock, 1 millimeter by default.

Delta X - stock shift along X axis. This parameter controls stock position relatively to the origin of coordinates of CNC-program along X axis.

Delta Y - stock shift along Y axis. This parameter controls stock position relatively to the origin of coordinates of CNC-program along Y axis.

Delta Z - stock shift along Z axis. This parameter controls stock position relatively to the origin of coordinates of CNC-program along Y axis.

Rotate Around X - rotate the stock around X axis on angle multiple of 90 degrees. Select value from dropdown menu.

Rotate Around Y - rotate the stock around Y axis on angle multiple of 90 degrees. Select value from dropdown menu.

Rotate Around Z - rotate the stock around Z axis on angle multiple of 90 degrees. Select value from dropdown menu.

Creating "StC" file

Create new CNC-program and load it in NCManager. Program the desired contour in usual G-code, following two rules:

1. program the contour in XY plane. Y sizes are linear, X sizes are diametral.

2. contour pass must be counterclockwise if you look on the contour from top.

Create stock file by calling Export > Stock contour from you CNC-program context menu.

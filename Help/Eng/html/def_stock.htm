Stock

To define a workpiece of the part being processed.

You right-click and the contextual menu containing the following items is activated.

All commands from the context menu depend on NCManager project type.

Create

To create a workpiece of the standard type.

There are two stock types in NCManager. Milling and lathe. Depending on the project type there are different Create commands in this menu. 

In milling project:

Contour

When stock will be defined, red sign to the left from "Stock" will disappear. Icons for quick activating will be changed. They are visibility control command and stock color command.

Delete

To delete a workpiece from the project.

Color

To change the color with the help of which a workpiece is displayed on the screen.

Visible

To hide or to show a workpiece in the course of processing.

Wire-frame

In the shade mode a workpiece is displayed as a wire-frame.

Transparency

To set the degree of transparency of a workpiece for the simulation mode and the shade mode

Section view - Set lathe stock section during simulation. This command is active in lathe and mill-turn projects only.

Edge - Show or hide stock edges.

Multi-colour - Enable or disable the possibility to paint stock into machine tool color.

Export/Import - Export and import of the stock.

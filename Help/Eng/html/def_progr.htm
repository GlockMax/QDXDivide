CNC-Programs

It contains a set of programs included in the project.

You right-click and the contextual menu is activated which contains the following items.

Load

Quick activating:

To load a file containing program text in the project. If the project has at least one program, the icon changes its color from red to green.

Create

To create a new program.

Renew

To load again texts of all the programs included in the project from the corresponding files.

Show All

To make toolpaths of all the programs in the project visible.

Hide All

To make toolpaths of all the programs in the project invisible on the screen. The icon changes its color to dark green.

Change Visibility

To make the hidden toolpaths visible and to hide the visible toolpaths included in the project.

Without Compensation

With Compensation

The corresponding mode being set, the program is displayed without/with compensation taken into account. Thus, if both of the modes are set, then the programmed toolpath and the toolpath with compensation are displayed simultaneously. Only the first mode is set by default, i.e. only the programmed toolpath is displayed.

Worked-out Toolpath

If the mode is set, only the toolpath section from the beginning to the current block is displayed. This mode helps to get the effect of the “wire-frame simulation”, i.e. the effect of the gradual drawing of the toolpath as the program is being executed.

If at least one of the programs exists in the project, a list of the loaded programs which you can View and Edit in the textbox appears when you double-click the left button.

You right-click any of the program and the Program contextual menu is activated.

Bookmark

In the Main Menu: Work > Bookmark

Quick activating:  icon.

This command enables or disables the possibility to stop the simulation or to move the machine tool in the bookmarked line. You can set the bookmark using  (Add/Delete Bookmark) command or using Find/Replace command choosing Select All.

For example, Bookmark is enabled ( icon is pushed), simulation mode is Smoothly ().Then if you push (To Breakpoint) button, the simulation process will be stopped after the bookmarked  line. Thus you can stop the simulation in any place you want.

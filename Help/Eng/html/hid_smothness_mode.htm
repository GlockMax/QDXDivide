"Mode > Smoothly" Command (Work Menu)

Quick activating:  icon.

This mode is used to get the machined stock solid model with cutting simulation in every line. For this mode you have to define Stock and Machine Tool. Simulation speed is adjusted by the slider that appears when you choose this mode.

In this mode you can instantly stop the simulation using pausecommand. When pause is pushed you can see the line-by-line simulation forward and backward on the defined (in the slider text box) number of lines. Moreover the forward simulation will be with the material deleting and backward simulation will be with material adding.

There are some restrictions in the Pause mode!

  • You can move forward and backward only for 30 lines from the place where Pause  button was pushed.

  • The speed that you change using a slider will be changed in fact only after the current line.

  • You cannot operate with Sections and Measures commands.

After you push Pause again the simulation will continue its work from the line where the Pause mode was turned on.

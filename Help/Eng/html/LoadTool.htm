Tools Loading

If the wireframe toolpath view is not enough - you can use the solid simulation. For CNC Program work simulation it is necessary to load tools and stock.

For quick creating of tools used in the CNC Program press right mouse button on Tools and choose From G-code.

This command will created all the tools used in the loaded CNC Programs. Tools list will appear in the Object Panel. To expand this list double click on Tools or click Cross button.  

This tools don't have any type or size. Now you can show or hide any toolpath by clicking respective check box. Icon next to this check box will help you to set tool's color.

To set tool's type and size press right mouse button on the tool's name and choose Edit. Appeared dialog box will allow you to choose tool's type and set its size and position. Tool's name will be automatically changed to show its size and type. You can also change tool's name to the one you want. Number 3 tool is the 6mm diameter drill with 100mm length.

First Tool - 8mm diameter end mill with 100mm length.

Second Tool - 2mm radius and 8mm diameter end mill with 100mm length.

Green line on the tool's name is showing that this tool is active and used in machining.

The last step to do is to create a stock.

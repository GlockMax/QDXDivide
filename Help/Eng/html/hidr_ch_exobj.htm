Fixture models

File, containing fixture model in STL format.

Right click in a model name to activate context menu.

Commands of the context menu act on the selected fixture model. Fixture model selected in fixture tree will be highlighted by its bounding box with a color of model.

Replace... - load another STL model in place of currently selected model. A standard file open dialog will appear.

Delete - remove selected fixture model from project.

Position... - shifting, rotation and scaling of selected fixture model.

Export... - save selected model to an STL file.

Properties... - show the "Properties" pane to view or set some fixture parameters.

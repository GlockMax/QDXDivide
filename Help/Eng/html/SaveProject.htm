Save a Project

In order not to repeat all actions connected with loading CNC Control, stock and some other parameters if you'll need to take a look at this program later, we have Projects in NCManager. Project file have an .NCM extension. All information about loaded CNC Programs, tools, stock and others is kept in project file.

File - Save command  will create or save already created project.

File - Load  command will load existent project.

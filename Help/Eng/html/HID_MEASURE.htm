Measure

In the Main Menu: Inspect > Measures

Quick activating:  icon.

Measuring the machined stock. This command is enable when one of the following modes is turned on - (Result), (Smoothly), (Quickly), in the active Stock window only. If the necessary mode is not chosen, you will see the Alert message:

When you push  icon, you will see the Measure dialog box:

Also the Measure tab in the Control panel will turn on automatically. In this tab you can find the information of type and parameters of a real surface that you are pointing on in the Stock window. Just move mouse cursor over the machined stock and look on the information in the Control panel. There are such surface types as cylinder, plane, torus etc. For more details, see Control Panel > Measure tab.

To measure the distance between any two stock walls that have metal inside them, the following mode should be turned on:

Then when pushing left mouse button program will measure the distance on the stock body direction. The yellow line will show you what was measured. Green ball is the start point, red - end.

In the Coordinates textboxes of a dialog box you will see the start point coordinates.

In the Direction textboxes of a dialog box you will see the direction vector. As indicated on the picture above this is the direction along the Y-axis. The direction is counted automatically by a normal to the surface in the chosen point, if the Constant Direction is not checked.

In the Result textboxes you will see the end point coordinates and the distance. In the example above it is L13.85.

You can set the start point coordinates and the direction on the screen by setting the values in the corresponding textboxes. In the example below it is the point with the 40,40,-5 coordinates and with direction along the X-axis. After you push (Execute), you will see the end point coordinates and direction. 

As in our example the point is "in the air" then in the Result textboxes you will see the distance between start point to the both sides of the given direction and the total distance. Also you will se the both end points coordinates.

You  can also set the constant direction checking the Constant Direction check box in the Measure dialog box. Then if you choose on of the points, program will measure the distance and the end point  coordinates using the constant direction. 

In this example direction is 1,1,0, i.e. 45 degrees.

To measure the distance between the stock edges with the "air" between them (grooves, apertures) you have to choose the mode shown below:

Then the  "to the air" direction distance will be measured. You will see the  yellow line that shows you what was measured. Green ball - start point, red ball - end point.

In each measure the real machined stock dimensions are measured that are independent of the stock model approximation accuracy during the simulation process.

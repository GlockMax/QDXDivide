Box

To set the dimensions of a rectangular workpiece.

Offset

To what extent to increase a workpiece length, breadth and height if they are calculated by overall dimensions.

Calculate by Overall Dimensions

A workpiece is set automatically by the size of all the feed motions toolpaths of all the programs being visible on the screen. Fast motions and hidden programs are not taken into account. A program›s zero is defined regarding the programmed point being indicated on the screen.

Length

To set a workpiece length in mm.

Breadth

To set a workpiece breadth in mm.

Height

To set a workpiece height in mm.

Tool path origin

It is the displacement relative to the programmed point indicated in the picture.

delta X- displacement in X-direction

delta Y- displacement in Y-direction

delta Z- displacement in Z-direction

Coordinate system

It is a point on a workpiece according to which the “floating zero” is set up.

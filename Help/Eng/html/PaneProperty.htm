Properties Pane

This pane is for fast change of properties and viewing of the help information at the specified element in Object panel.

CNC-Control - Properties for initial state of CNC-Control object.

Machine - Properties for mashine tool.

Machine parts - Properties for each part of machine tool.

CNC-Programs - Properties for all programs

Program - Properties for each program.

SubPrograms - Properties for all subprograms (Not properties)

SubProgram - Properties for each subprogram..

Tools - Properties for all tools.

Tool - Properties for each tool.

Stock - Properties for stock

Parts - Properties for all parts.

Part Properties for each part

Fixture Properties for all fixture parts.

Fixture part Properties for each fixture part

Contour

Set stock using contour. Contour is defined in the "StC" extension file. Z-axis is the pivot pin.

Find

This command is used to load the file defining stock geometry. Open dialog box will be activated with the *.StC extension of the loaded files.

Delta Z

Stock moving on the Z-axis. You can change the stock position comparing to the coordinate system origin.

When you load the stock you will see its Dimensions.

 

How to create a "StC" file.

Create new program and load it into NCManager. Set the contour in the G-code format. Observe this two rules.

1. Set contour in the XY-plane. X-axis size will be diametral, while Y-axis size will be linear.

2. Direction must be CCW if you look down on the contour.

Create stock file (Export > Contour) from your program's context menu.

Tool Change

In the Main Menu: Work > Tool Change

Quick activating:  icon.

This command enables or disables the possibility to stop the simulation or to move the machine tool in the line that goes after the Tool Change command.

For example, Tool Change is enabled ( icon is pushed), simulation mode is Smoothly () and there is more than one NC program with several machine tools in the NCManager project. Then if you push (To Breakpoint) button, the simulation process will be stopped after the line of tool change. Thus there will be an automatic simulation stop after the end of the each machine tool's work.

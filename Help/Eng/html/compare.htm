Comparison function

Comparison function allows to figure out the difference between the stock and the part.

To call the comparison function select Analysis > Comparison from main menu.

To use comparison function the part should be loaded and the program should be in "Cut" simulation mode.

When called, the comparison dialog appears. The calculation begins immediately. The progress bar at the bottom of the dialog indicates calculation progress. Calculation time strongly depends on the complexity of the part and the stock.

The bounds values define the borders between the deviation ranges in millimeters. The values must be in ascending order from top to bottom. This values are stored in the project file.

Each range of deviation is assigned a color. For example, if some part of the stock surface lies in a range between 0.01 mm and 0.5 mm distance from the part surface it will be colored green (see the figure above). All the options for this analysis is stored in a project file.

Default colour - reset color settings to defaults.

Default bounds - reset bounds to defaults.

Apply - make the analysis with new colors and bounds.

When the analysis is made, a new stock coloring is shown.

To exit the comparison mode close the comparison dialog by pushing the close button in the upper right corner of the dialog.

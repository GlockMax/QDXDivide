Tool properties

Some properties of selected tool in Object pane are managed from Properties pane.

To change property, click the desired property value in the right column, then select new value from list or enter value from keyboard. Modified parameter will be highlighted bold. There's a hint for currently selected property at the bottom of the pane.

Toolpath - controls toolpath visibility (in all cnc-programs) for currently selected tool. "No" for invisible, "Yes" for visible.

Quick toolpath visibility toggle:

click on the lamp icon in object panel for corresponding tool.

 

Wireframe - controls display mode for currently selected tool and its holder. "No" - tool is displayed solid with selected color, "Yes" - tool is displayed as wireframe with selected color. Wireframe step is set in Settings > Parameters > Graphics "Tool Mesh", and it can be saved in settings file (*.dfn) using Settings > Save settings...

Cut length - controls tool's cutting surface display color.

Quick cutting surface color selection :

click on the colored square in object panel for corresponding tool .

 

Clear length - controls tool's non-cutting surface display color.

There are two colors a tool have - cutting surface color and non-cutting surface color. If Stock multi-color mode is on, then the surface of stock formed by a tool would have the corresponding color of cutting surface of this tool.

Example for milling tool.

For lathe tools the cutting surface color is applied to the cutting edge of a turning plate. If Stock multi-color mode is on, then the surface of stock formed by a plate would have the corresponding color of cutting edge of this plate. If the turning plate was cutting the stock with non-cutting surface, then the surface of stock formed by a plate would have the corresponding color of non-cutting surface of this plate.

Example for lathe tool .

Holder - controls tool's holder color.

Tool - set up the tool number in a tool bank. The tool is referenced by this number from CNC-program.

Axis - set up the axis number for the tool. This is a special concept, made to make it possible to create a tool with any angle to the spindle. Axis position and number are set in the machine description file.

Turret - tool bank number. The number should be defined in machine description file.

Marker - Defines the size of tool marker. When a tool is hidden or a tool size and shape are not defined, tool's current position is marked by this marker. Default marker size os set in Settings>Parameters>Graphics>Tool Marker.

Time - displays the tool working time.

Fast motions length - displays the length of no-load running.

Feed motions length- displays the length of feed running.

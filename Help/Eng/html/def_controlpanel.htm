Information panes

Information panes are used to display various parameters of the project.

Background color, text size and color settings of the information panes may be changed in Settings > Parameters on a Design pane . Select ControlPanelBackground or ControlPanelText from the Element combo box.

Information panes contains four panes:

Parameters pane

Parameters pane is used to display the geometrical, technological and other parameters, changing during program execution.The parameters are displayed for the current program line.

To add or remove parameters to be displayed, choose the Options… from pane 's context menu or Settings > Parameters > Parameters pane from main menu.

Geometry pane

Geometry pane displays geometrical parameters of toolpath element, highlighted by the mouse cursor in HIghlight mode (View > Highlight).

For a segment it shows the coordinates of its ends, angles between the segment and coordinate axes, and its length.

For an arc the pane therewith shows the arc radius and its center coordinates.

For a point the pane shows its coordinates and angles between previous and next CNC program lines.

 

Measurement pane

Measurement pane displays additional information about the stock surfaces, their type and parameters. It works when the Measure dialog is active (Inspect > Measurement).

To view the surface parameters move the mouse cursor on the target surface, and the pane will display them.

For the stock (crude parts) the pane shows the coordinates of a point on the stock surface.

For a plane the pane shows its normal coordinates and the coordinates of a point on the plane.

For the cylinder the pane shows whether it is convex or concave, center coordinates, axis direction and cylinder radius.

For torus the pane shows whether it is convex or concave, center coordinates, axis direction and two torus radiuses.

For the sphere the pane shows center coordinates and radius.

Registers pane

Registers pane displays the values of registers contained in CNC program. If the register value is zero, this register is not displayed. If the register value changed during execution of current CNC program line, this value is highlighted red. Otherwise, the value is black.

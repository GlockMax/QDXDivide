Program View and Editing

The next step of the work with NCManager is the addition of NC program presented in G-codes. A certain command that sets this or that machine operation corresponds to every line of this program. You can either Create a new or Load an already existing program.

Program Creation

You can create a program by selecting the Create item on the menu that appears on right-clicking the Programs item on the Object panel. In the opened dialog you should create a filename and a catalog where you will place the program being created.

Remember that a newly created program is not added automatically into the project. To add it you should Load.

Program Load

To load a program you should select the Programs item on the Object panel, right-click it and select the Load... item on the menu appeared.

Remember that if you work with NCManager demo version, you will not be able to load your program.

Display of the Loaded Program

In order to view the text of the loaded program you should double-click the left button on the selected program. This program will be displayed in the textbox.

While changing the loaded program you can control the changes made by displaying them in the graphical window. You can do this by selecting the Apply item (on the contextual menu that appears when you right-click the textbox) or pressing Ctrl+R keys.

Moreover, you can view and edit the program text. There are additional functions in the textbox.

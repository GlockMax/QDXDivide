Operating with Views

In order to operate with views (to have an opportunity to examine the picture), press right mouse button on a graphic window. Appeared toolbar consist of 24 buttons that will help you to operate with image size and to arrange image position on the screen. Click the Show All icon.

You will see the toolpath in the Stock window:

You can make the same actions using View tab in the Main menu. Also you can find some of the items in the View toolbox.

Elements of this menu are enabled if the graphic window is active. Therefore if you see that necessary function is disabled you just need to click on the graphic window.

The UCS icons intensity level will tell you whether one of the windows is active or not:

You can use a very comfortable way of dynamic control by pushing and holding Ctrl, Shift, and Alt buttons together with left mouse button (LMB). Ctrl+LMB initiates dynamic rotation; Shift+LMB is used for panning or to move the image; push Alt+LMB to change the scale or to change the image size.

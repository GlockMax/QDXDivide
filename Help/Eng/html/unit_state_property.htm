Control logic rack properties

Some parameters from Initial state CNC-control dialog are placed in a Properties panel. These parameters correspond to initial parameters of control logic rack and affect the execution of CNC programs in NCManager.

To change these values, click in the left column of Properties panel on a desired parameter and select its value from the list or enter from keyboard. Modified parameter will be highlighted bold. Parameter description is displayed at the bottom of the panel.

Initial tool number and coordinates from program execution are represented by following parameters:

Tool - the number of tool in a tool bank, which is going to machine. If the CNC program doesn't have a command to change the tool, the machining will be carried out with this tool. Otherwise, tools are used according to Object panel >Tools.

Turret - the number of a tool bank, which contain the tool with number set in "Tool" property.

Start X - initial tool position by X linear coordinate.

Start Y - initial tool position by Y linear coordinate.

Start Z - initial tool position by Z linear coordinate.

Start A - initial tool position by A angular coordinate.

Start B - initial tool position by B angular coordinate.

Start C - initial tool position by C angular coordinate.

Initial values of some parameters or default logic control rack settings are set by:

Feed - tool advance measuring units (mm/min, mm/rev).

Feed value - tool advance value.

Chuck direction - rotation direction of main spindle (CW, CCW).

Chuck speed - main spindle rotation speed value.

Programming mode - coordinates programming mode (Absolute mode, Incremental mode).

Unit measure - coordinate measure units (millimeter, inch).

Debug - switch debug frames execution (i.e. frames beginning with “/”). To turn it on select "Yes", otherwise - "No".

File - logic control rack properties file location.

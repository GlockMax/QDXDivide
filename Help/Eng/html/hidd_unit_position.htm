Cutter ID

Tool start

The coordinates of the tool before processing according to the program.

Tool number

The position number of the tool used in processing. If the program lacks a command which changes tools, then processing will be done with the tool which position has been set here. If there is a command for changing tools in the program, then in the course of processing the tools defined on the Tools menu will be taken.

Debug

To start or to stop the execution of the debugging blocks (for example, starting with the “I” symbol). ). In order to set this mode you should select the Debugging check box. In order to unset it you should clear the check box.

"Work" Commands

Extra Check

Extra possibility to check G-code on length compensation and presence of IJK in the circular interpolation line.
Mode

Set one of four simulation modes:

Result

Smoothly 
Quickly
Wireframe
Go To

Put the Machine Tool to the defined place: 

Backward
Forward
N Lines Backward
Execute

Show the machine tool moving process:

To Breakpoint
N Lines

Reset

Stops the simulation, returns the initial stock and goes to the beginning of the project.
Pause

Instantly stops the simulation in the Smoothly mode.
Stop

Stops the simulation. Stop line becomes current.

Current Stock

Show the work result till the current place.
Initial Stock

Returns to the initial stock, current place remains the same.
Breakpoints

Set the possibility to stop the simulation in the defined places of the CNC program.
Program Start/Stop

Simulation will be stopped on every start and end line of each CNC program of the project.

Optional/Program Stop

Simulation will be stopped in the lines with the optional stop commands. Usually they are M0, M1 codes.
Tool Change

Simulation will be stopped on the line of tool changing.
Bookmark

Simulation will be stopped on the bookmarked line.
Extra Breakpoints

This command turns on or off the possibility to stop the simulation in extra breakpoints.

Add standard tool

This command is used to add end-milling cutters, drills and set their parameters.

Select appropriate tab to set the tool type.

Common parameters:

Diameter (D) - tool diameter.

Height (H) – total height of the tool(tool overgang).

Tool cut surface – you can set cut surface length. To set cut surface length uncheck Whole Tool checkbox and type height value in the Height (h) field. You can also set different colors for cutting and noncutting surfaces of the tool.

Name - tool name is formed automatically according to its type and size, but you may set it manually.

Tool-specific parameters

On the first (leftmost) tab you can set parameters for end-milling cutter without rounding.

Additionally you may set the chamfer. To set the champher parameters check "With chamfer" checkbox and fill in its parameters:

Angle (a) - chamfer angle

Chamfer - chamfer length.

On the second tab you can set parameters for end-milling ball cutter.

To set particular rounding radius uncheck the "With ball end" checkbox and fill in the radius.

Radius (R) - rounding radius can be greater than tool radius.

On the third tab you can set parameters for cone end-milling cutter.

You can additionally set:

Radius (R) - rounding radius.

Angle (a) - tools cone angle.

On the forth tab you can set parameters for drilling tool.

Angle (a) - drilling tool angle. Default value is 118 degrees.

Geometry pane

Geometry pane displays the geometrical properties of selected toolpath element when in highlight mode (main menu View>Highlight).

You can toggle pane visibility from main menu Settings > Panes.

You can change the pane position on screen, dock it to any other pane, make it autohide.

Background color, text color and font size for geometry pane are set in Settings > Parameters... > Design. ФонПанели или ТекстПанели.

If a segment is selected - start and end point coordinates, angles between the segment and coordinate axes, and length are displayed.

If an arc is selected - in addition the arc center and radius are displayed.

If a point is selected - it's coordinates and the angle between previous and following cnc-program line are displayed.

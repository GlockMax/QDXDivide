Feedrate tool motion

Main menu : Work > Tool motion mode > Feedrate

Shortcut: icon .

In this tool motion mode the tool moves smoothly along the toolpath. Tool speed corresponds to the feed rate specified in CNC-program. Movement speed can be increased or decreased by a track bar.

If you're in a cutting mode, this way of tool motion allows to control stock cutting inside each CNC-program line.

In this case you can immediately stop the simulation by pressing pause . When pause is pressed, it is possible to move simulation process forward or backward by a number of steps set up in the input field. So the simulation process may be moved forward with material removal, or backward with adding material.

There are lsimulations in pause mode!

  • You can move only 30 lines forward or backward from the line you pushed pause.

  • After changing the speed of simulation, actual speed will change only after the line where it was changed will be simulated.

  • You can't use Section or Measurement

When you exit the pause mode, simulation will continue from the line where pause was pressed.

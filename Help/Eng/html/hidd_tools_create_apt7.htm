APT 7 standard tool creation dialog

A milling tool can be set by APT 7 standard parameters.

APT7 standard parameters:

1. Tool diameter D.

2. Tool rounding radius R.

3. Length E. Distance from tool axis to the center of rounding along X axis.

4: Length F. Distance from top plane of tool to the center of rounding along tool axis.

5: Angle b at the bottom of a tool.

6: Angle a at the side of a tool.

7: Tool height H.

Majority of tools can be set through APT7 standard. This type of tool is mainly used to set end-cutting tools and drilling tools from external file in special JOB format. You can use JOB file to pass all project parameters from any external application.

End-milling cutter without roundings

Diameter D> 0

Height H> 0

Radius R = Angle a = Angle b = 0

End-milling cutter with roundings

Diameter D> 0

Height H> 0

Radius R > 0 < D/2

The rest of the parameters = 0

End-milling cutter with ball-end

Diameter D> 0

Radius R > 0 = D/2

Height H> 0

The rest of the parameters = 0

Drilling tool

Diameter D> 0

Height H> 0

Tool angle b.> 0

Radius R, Tool angle a = 0

The rest of the parameters = 0

End-milling conical cutter with ball-end

Radius R = D/2

Tool angle a > 0

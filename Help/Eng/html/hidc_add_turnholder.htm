Creating turning toolholder

This dialog allows you to set toolholder dimensions and position.

Rectangular toolholder parameters (Bar):

 

X – toolholder length along X axis

Y – toolholder length along Y axis

Z – toolholder length along Z axis

D – toolholder orientation on XZ plane

Offset X – toolholder offset along X axis

Offset Z – toolholder offset along Z axis

Offset Y – toolholder offset along Y axis

Round toolholder parameters (Cone):

X – toolholder length

D1 – toolholder left diameter

D2 – toolholder right diameter

Offset X – toolholder offset along X axis

Offset Z – toolholder offset along Z axis

Offset Y – toolholder offset along Y axis

D – toolholder orientation on XZ plane

Full diameter – the toolholder will be full cylinder/cone.

Half diameter – the toolholder will be a half of cylinder/cone bounded by plane going through cylinder/cone axis.

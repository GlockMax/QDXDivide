"Extra Check" Command (Work Menu)

The extra possibility to check G-code on:

  • turning on the machine tool length compensation

  • compulsory presence of I, J, K, R in the circular interpolation line

  • conformity of machine tool radius compensation number with the machine tool number

  •  conformity of machine tool length compensation number with the machine tool number

If you don't need such a possibility - switch the check box off.

 

The Extra Check is turned on.

If the first Z-axis move will be met in the NC program after the machine tool change while the length compensation will be turned off, program will show you the Error Message and the corresponding line will be marked with the yellow cross:

If there are none of the addresses (I, J, K, R) met in the line during the working circular interpolation, then the Error Message will be shown and the corresponding line will be marked with the yellow cross:

If the machine tool diameter compensation number is not equal to the machine tool number plus integer that was set in the CNC control file, then the Error Message will be shown and the corresponding line will be marked with the yellow cross:

If the machine tool length compensation number is not equal to the machine tool number plus integer that was set in the CNC control file, then the Error Message will be shown and the corresponding line will be marked with the yellow cross:

Every check can be adjusted in the CNC control description file. Thus it is possible to turn on one (or all) of the checks. 

If the Extra Check command is turned off then no check will be realized no matter what is indicated in the CNC control description file.

View CNC Program Text

You can view and edit CNC Program text in the following way. In this case there is only one CNC program.

Double click CNC-Programs on Object Panel or click "+" on the left of CNC-Programs and you will see the list of all loaded programs.

Load CNC Program text into the Text Window by double clicking the name of the program you need (demo.nc).

You can also choose the file name item (demo.nc) then push right mouse button and click Text.

You can view and edit the CNC program text in the following way. Double click on CNC-Programs. You will see the list of loaded NC programs. Then double click on the required program name and the program text will be loaded into the Text Window . The name of the NC program which text is loaded in the textbox appears bold. After that you can simply edit this text. In order to apply the changes click right mouse button and choose Update.

The name of the CNC program which text is loaded in the Text Window appears bold. You can change window size using your mouse. Also you can change Text Window position at the bottom or to the left, or using Main menu (Window > Configurator) or with the  icon from the toolbox.

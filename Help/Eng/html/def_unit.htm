CNC-Control

CNC-Control defines the information necessary for examining the work of the NC program of the machine with CNC.

You right-click and the contextual menu is activated which contains the following items.

Load

Quick activating:

It loads the *.sm* file in which the G codes and other information necessary for correct reading of the program at a certain machine are defined. Load the CNC you will deal with. The icon will change its color from red to green which means that the CNC has been set.

Start State

Quick activating:

CNC initial status is the value of the parameters which are set before the program executes. You can set:

Cutter ID

Speed

Mode

Compensators

Work coordinate systems

Toolbars

Toolbars provide fast access to the main functions.

Standard toolbar provides access to the basic items of File menu.

View toolbar provides access to the basic items of View menu.

Modes toolbar allows to switch between simulation modes (Work menu).

Navigation toolbar controls tool movement according to the program (Work menu).

Breakpoints toolbar provides breakpoints management in the CNC program (Work menu).

Inspect toolbar provides fast access to the basic items of Inspect menu.

View toolbar provides control over the graphical windows appearance (Window menu).

Step_speed toolbar controls the speed of simulation and tool movement according to the lines in the CNC program (Work menu).

Indicator toolbar displays current CNC status.

RAPID: RED - noncutting stroke, GREEN - cutting stroke.

COOL: GREY - cooling is off, GREEN - cooling is on.

SPEED: GREY - spindle is off, GREEN - spindle is on.

SUB: GREY - main program is running, GREEN - subprogram is running.

CYCLE: GREY - loop (machine cycle) is off, GREEN - loop is on.

COMP L: GREY - length offset is off, GREEN - length offset is on.

COMP R: GREY - radius offset is off, GREEN - radius offset is off (cutter compensation, cutter offset).

TOOL: GREY - no instrument loaded, GREEN - instrument loaded.

Editor. Provides access to basic items of Edit menu .

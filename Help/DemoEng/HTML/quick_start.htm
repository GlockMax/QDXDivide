Quick start
 

After starting NCManager, it automatically loads KorpDemo.ncm project.

Demo projects are in NCManager folder, ..\Demo\Millimeter\. You can load any project by calling main menu FIle > Open...

Here we describe the the basic operation to start smooth machining simulation in NCManager.
 
Step 1

Switch to Cut mode

  • Click the icon in toolbar
 
Step 2

Start cutting

  • Click the icon in toolbar
 
 

You can adjust simulation speed with the track bar
 

You can control graphical window:

Left button pressed - rotation
Middle button pressed - translation
Mouse wheel - scaling
Right click - view control context menu

 
 

You can control the visibility of the system windows:
 

Machine - in this window you can see the wireframe toolpath and you can see the movement in accordance with cinematic machine tool scheme.

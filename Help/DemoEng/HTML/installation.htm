Program Setting

Program Installation

1.In order to install NCManager, run Setup and follow the instructions which will appear on the screen. The shortcut which helps to run the program will be placed in the Programs item of the Search menu.

2.On accomplishing the installation insert the protection key and run NCManager.

Program Removal

In order to remove the installed NCManager program use the Windows features, namely, the toolbar and the program Add or Remove Programs. The mere removal of the installed files will be incorrect.

The first steps in NCManager demo

Run NCManager and load Demo.ncm project file.

Main menu: File -> Open … .

If this is the first time you run NCManager it will look like this:

The Objects pane is designed for displaying and controlling the project main items

CNC-Control defines the information necessary for examining the work of the NC program of the machine with CNC.

Machine defined the CNC machine tool.

CNC programs It contains a set of programs included in the project.

Tools It contains a set of cutting tools included in the project.

Stock To define a workpiece of the part being processed.

Parts contains a set of the STL models loaded into the project as parts.

Fixture - contains a set of the STL models loaded into the project as fixture.

Left click mouse button on icon of the Object Pane:

To load corresponding object

To create corresponding object

To edit parameters CNC-Control

To switches a mode of display of a trajectory (show all or a part)

To hide or show cutting tool

To hide or show corresponding object

To define color corresponding object

Right click mouse button on any word of the Object Pane makes active the contextual menu.

View control

In order to operate with views (to have an opportunity to examine the picture), press right mouse button on a graphic window. Appeared toolbox consist of 24 buttons that will help you to operate with image size and to arrange image position on the screen.

You can make the same actions using View tab in the Main menu. Also you can find some of the items in the View toolbar.

You can use a very comfortable way of dynamic control by pushing and holding Ctrl, Shift, and Alt buttons together with left mouse button (LMB). Ctrl+LMB initiates dynamic rotation; Shift+LMB is used for panning or to move the image; push Alt+LMB to change the scale or to change the image size.

Additional view controls!
Shortcut for Pan: Click and drag the Middle mouse button
Dynamic Zoom Mouse with Middle Roller - Push / Pull

analyseing toolpath

While you move your mouse over the toolpath image CNC program lines and nodes will be highlighted. At the same time cursor shape is showing you the element type and the toolpath direction

Attention! Stock Window must be active.

The geometrical characteristics of the highlighted element can be viewed on the Geometry pane.

The coordinates of the object pointed by the cursor, the block number and the program name are displayed at the bottom of the screen.

 

Three numbers on the right are the coordinates relativly to the CNC program "0" point. Next to these numbers on the left there are another three numbers showing us coordinates relatively to the cursor position (you can change this "0" point by simply clicking somewhere on the Graphic Window). Number in the brackets shows the distance.

Left mouse button click on any toolpath part will lead to loading of the corresponding CNC program into the Text Window and to the pointing on the corresponding line.

Double click will cause program run until current line. This is how you can find out the time of program run until current line or any other parameters of the current CNC control condition which are available from Parameters pane.

You can set the number of displayed parameters of the Parameters pane from main menu Settings > Parameters … > Control panel or from the contextual menu of the Parameters pane.

View CNC Program Text

Double click CNC-Programs on Objects pane or click "+" on the left of CNC-Programs and you will see the list of all loaded programs.

Load CNC Program text into the Text Window by double clicking the name of the program you need (demo.nc).

You can also choose the file name item (demo.nc) then push right mouse button and click Text.

The name of the NC program which text is loaded in the textbox appears bold.

After that you can simply edit this text.

The text editing commands are in the context menu of the Editor pane. Most commands are duplicated in the Editor pane’s toolbar.

If the CNC Program text was modified and you want to save the changes - push right mouse button in the Editor pane and click Update.

Tools Loading

For CNC Program work simulation it is necessary to load tools.

Double click Tools on Objects pane or click "+" on the left of Tools and you will see the list of all loaded tools.

Now you can show or hide any toolpath by clicking respective check box. Icon next to this check box will help you to set tool's color.

To set tool's type and size press right mouse button on the tool's name and choose Edit. Appeared dialog box will allow you to choose tool's type and set its size and position.

Create a Stock

To create or change a stock choose Create item in the Stock context menu and set stock's type and size.

Work Simulation

In order to check or to gain an understanding of a specific NC program (G-code), you can find two simulation modes in NCManager “Cut” and “Don’t cut”.

Cut - in this mode you have an opportunity to make a stock cut simulation.

Don’t cut - toolpath view in the form of lines and arcs with no stock cut.

Both Cut and Don’t Cut modes my be in one of four motion modes. Motion mode can be changed by the main menu Work or by the toolbar icons.

Feedrate - When this motion mode is active the tool moves smoothly along the toolpath. The speed of the tool motion corresponds to the federate defined in the CNC program. Tool motion speed can be changed by the toolbar slider.

Smoothly – When this motion mode is active the tool moves smoothly along the toolpath. Tool motion speed can be changed by the toolbar slider.

Quick - When this motion mode is active the tool position is shown by the given time interval. Tool motion speed can be changed by the toolbar slider.

Result - When this motion mode is active no tool motion is displayed.

Tool motion control commands.

Show the work result till the current place

Returns to the initial stock, current place remains the same.

Show the machine tool moving process till the nearest breakpoint or program end.

Move back to the previous breakpoint or program start

Move forward to the nearest breakpoint or program end

Pause

Stops the simulation. Stop line becomes current.

Reset. Stops the simulation, returns the initial stock and goes to the beginning of the project.

Tool motion control commands for given lines numbers and motion control speed slider.

To view the stock cutting with the smooth tool motion it is necessary to:

Turn on Cut mode.

Turn on Smoothly motion mode.

Press Reset icon to returns the initial stock and go to the beginning of the project.

Press To breakpoint icon.

NCManager will now start simulation process of deleting stock material. To stop the simulation push Stop button. After the end of simulation you will see the following:

Examine the finally machined stock by changing the view and enlarging parts that you are interested in.

Section command’s icon. You can build sections in Graphic Windows.

Measure command’s icon . You can taking measurement only in Stock window.

Load project and start simulation (mill project)

In this section we will learn how to:

·         Load an existing NCManager project;

·         Hide all the toolpaths of CNC programs;

·         Switch the cutting simulation (used to realistically simulate the cutting process) on and off;

·         Control the simulation process by commands "To breakpoint", "Stop", "Pause" and others;

·         Switch between simulation speed modes: «Smooth», «Result», «Quick» and «Feedrate»;

·         Jump to a desired line of CNC program by double clicking a program line in NCManager CNC program editor window;

·         Control the view in graphical window by rotating, shifting and zooming with a mouse or using View toolbar.

NCManager demo version automatically loads KorpDemo.ncm project on start.

A project is an NCManager file with *.ncm extension, which contains objects needed to verify CNC program execution process. Those objects are CNC program files, cutters specifications, stock model, CNC control options and so on. An NCManager project does not necessarily contains the full set of objects.

Demonstration projects could be found in NCManager installation directory, ..\Demo\Millimeter\. Any of these projects could be loaded using the main menu command File > Open... .

Open the project AcromatikDemo.ncm

Let's go through the major steps to start a smooth cutting simulation in NCManager.

Hide all toolpaths by clicking the icon with a "lit lamp" to the left of CNC-Programs items group in Objects pane. Point out that the icon has changed to "dark lamp".

All the toolpaths of all tools in project are now hidden. The graphical window now shows current tool model and stock wireframe model.

Switch to the "Cut" mode. To do this, unfold the drop list of cutting modes in the "Modes" toolbar and click the "Cut" item.

Then the list will collapse and the icon of the list would look as follows:

In the "Cut" mode the stock model in graphical window will be displayed solid.

Switch the tool movement mode to "Smooth" by clicking the following icon in "Modes" toolbar.

Start the cutting simulation by clicking "To breakpoint" button in "Navigation" toolbar.

You can control the cutting simulation speed with the slider on a "Step speed" toolbar from left (low speed), to right (high speed).

You can control the view in graphical window with your mouse whether a simulation is running or stopped:

·         Left mouse button hold – rotation

·         Middle mouse button hold – transition

·         Mouse wheel - zooming in and out

·         Right mouse click in graphical window shows the view control panel

To immediately stop the simulation click on "Pause" button or press F5 key.

As soon as you switch to "Pause" state, the simulation would stop. The green arrow in text editor pane now points to the line which is now being executed.

 You can simulate a desired number of lines forward or backward from current position by using the buttons on "Step speed" toolbar when simulation is in "Pause" state.

To continue the simulation from "Pause" state, just click on "Pause" button again, or press F5 key.

To stop the simulation click the "Stop" button on "Navigation" toolbar or press F6 key.

Simulation immediately stops, but the stock would be shown in a currently calculated state, i.e. in a state where the stock is machined further than you saw during simulation right before you pressed "Stop".

To return to initial simulation state (initial stock model, beginning of CNC program), click on "Reset" button on "Navigation" toolbar.

You may execute the "Reset" command at any time. Now let's execute it.

Now we will discuss partial simulation to a particular NC program line. Find the line number 1000 in the text editor pane and double click on this line. Now this line became the current line of NC program. There is a blue arrow marker to the left of the line, which indicates current NC program line.

Start partial simulation to the current line by pressing the following button on a navigation toolbar.

The simulation will start from initial state (the beginning of NC program, the stock will be reset to its original form), simulation speed mode will remain the same, i.e. "Smooth". When the simulation reaches line number 1000 it will immediately stop.

Wait till the simulation stops, then switch the simulation speed mode to "Result" by clicking the following button on the "Modes" toolbar.

 

Start partial simulation to the current line again. After some period of calculations, the graphical window would display the stock model machined to the line number 1000. No cutter movement is shown in this simulation speed mode.

Now You may repeat partial simulation to the current line for two other simulation speed modes - "Quick" and "Feedrate". 

To switch to the Quick simulation speed mode, click on the following button on "Mode" toolbar.

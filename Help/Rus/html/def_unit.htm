Стойка ЧПУ

Определяет необходимую информацию для просмотра работы управляющей программы на станке с ЧПУ.

При нажатии правой клавиши мыши на слове Стойка ЧПУ, активизируется контекстное меню, содержащее следующие пункты.

Загрузить - загрузить другую стойку ЧПУ.

Исходное состояние - задать значения некоторых параметров стойки ЧПУ через диалог.

Свойства - задать значения некоторых параметров стойки ЧПУ через панель "Свойства".

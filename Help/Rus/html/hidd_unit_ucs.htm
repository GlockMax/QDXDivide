Системы координат

Для изменения значения смещения осей координат для G-кода необходимо указать мышью в нужное поле и поставить значение.

Для видимости значка системы координат в графических окнах, нужно поставить галочку в поле "Видимость".

Видно если есть значение – если опция выбрана, то значок СК будет показываться, если есть не нулевые смещения и включена видимости значка.

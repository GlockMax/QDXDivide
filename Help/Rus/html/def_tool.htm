Инструмент

Этот пункт меню служит для управления параметрами конкретного инструмента. Нажатие правой клавиши мыши на имени инструмента вызовет контекстное меню инструмента:

Изменить - задать новые параметры инструмента.

Изменить ... фрезерный инструмент
Изменить ... токарный инструмент

Вырезать - поместить параметры инструмента в буфер обмена.

Копировать - копировать параметры инструмента в буфер обмена.

Вставить - создать новый инструмент из буфера обмена.

Удалить - удалить инструмент из списка инструментов.

Свойства - показать панель "Свойства", чтобы задать или посмотреть значения некоторых параметров для указанного в панели Объекты инструмента.

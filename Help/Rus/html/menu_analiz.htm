Команды меню Анализ

- Некоторые команды вынесены в панель инструментов Анализ.

В меню находятся команды для анализа работы программ:

Столкновения - Задать параметры контроля столкновения станка.

Сечение - Динамическое сечение заготовки.

Поставить Размеры - Построения размерных и геометрических примитивов для измерения любого объекта

Измерения - Измерения заготовки.

Статистика программ - Посмотреть информацию о времени работы программ.

Сравнение - Сравнить обработанную заготовку с моделью детали и выделить отличия заданными цветами.

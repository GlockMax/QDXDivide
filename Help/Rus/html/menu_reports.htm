Команды меню Отчеты

Команды служат для создания, активации и просмотра отчетов о работе программы

Создать/Редактировать - создать новый или изменить существующий отчет. При нажатие на команду вызывается диалог Редактор отчетов:

Поле Отчеты - Cодержит список ранее созданных отчетов.

Новый – Создать новый отчет. Нажмите на кнопку Новый, чтобы начать создавать новый отчет

Удалить – Удалить выбранный отчет. Укажите мышкой на удаляемый отчет в поле Отчеты и нажмите кнопку Удалить. Отчет будет удален.

Сохранить – Сохранить все параметры созданного или измененного отчета.

Выход – Выйти из диалога Редактор отчетов.

Поле Название служит для ввода имени отчета. Под этим именем будет создана закладка в диалоге Просмотр отчетов.

Поле Описание служит для справочного описания того, что делает этот отчет.

Условие – В этой группе помещены поля и команды для задания условий, по которым в отчет будет записываться информация, заданная в таблице Поля таблицы.

Условие отбора – В этом поле пишется условие отчета. Под условием понимается фраза типа: Если Параметр равен чему-то. Список возможных параметров находится в таблице Параметры. Чтобы добавить параметр в поле Условие отбора, выделите его мышкой и нажмите на иконку <=. Параметр можно написать вручную и удалить в поле Условие отбора. Чтобы добавить логические условия, арифметические операции логические операции и приоритеты, можно использовать иконки Операции или набрать эти символы вручную.

Операции

«+ - * /» - Арифметические операции: сложение, вычитание, умножение, деление.

«< > = !» - Логические операции: меньше, больше, равно. ! (восклицательный знак), задает условие неравно.

«| &» - Логические операции «Или» и «И».

«[ ]» - Открывающая и закрывающая скобки.

Возможные поля – Содержит список возможных параметров, которые могут быть помещены в отчет. В отчете может быть несколько полей (столбцов). Каждое поле определяется параметром, который будет в него помещен. Чтобы добавить параметр в поле Поля таблицы, выделите его мышкой и нажмите на иконку <<.

Поля таблицы – Содержит список параметров, которые буду в задаваемом отчете. Чтобы удалить параметр из поля Поля таблицы, выделите его мышкой и нажмите на иконку >>. Параметры будут помещены в порядке заданном в таблице сверху вниз так, что самый верхний будет левым крайним в отчете. Порядок параметров в таблице Поля таблицы можно менять, выбрав параметр мышкой и нажимать на иконку вверх вниз (иконки с жирными стрелками вверх), чтобы переместить выбранный параметр выше или ниже.

Выбрать - выбрать из существующих отчетов, нужные и сделать их активными. При нажатие на команду вызывается диалог:

Выберите нужный отчет в поле Неактивные и нажмите на иконку «=>», чтобы добавить отчет в поле Активные.

Выберите нужный отчет в поле Активные и нажмите на иконку «<=», чтобы удалить отчет из поля Активные.

Очистить отчеты - удалить выбранные активные отчеты. При нажатие на команду содержимое всех активных отчетов очистится:

Посмотреть отчеты - вывести на экран результаты работы отчетов. При нажатие на команду вызывается диалог:

Нажимайте закладки вверху с именами активных отчетов, чтобы посмотреть содержимое этого отчета

Сохранить – Сохранить в текстовый файл содержимое отчета.

Выход – Выйти из этого диалога.

В поставке NCManager, как примеры, включены два отчета.

1. Короткие кадры

2. Выход за габариты

Чтобы посмотреть отчет нужно:

Командой Выбрать поместить требуемый отчет в поле Активные. Для этого нужно поместит курсор на нужный отчет в поле Неактивные и нажать на иконку =>. Нажать на кнопку OK для выполнения и выхода из команды Выбрать.

Далее выполнить программу. Например, для быстрого получения результата в режиме Каркас нажать на иконку Перейти к следующей ТО

Командой Посмотреть отчеты вызывается окно, где помещены результаты работы выбранного отчета при выполнении программы. Каждый активный отчет помещен под закладкой с именем отчета. Нажмите на нужную закладку, чтобы посмотреть содержания соответствующего отчета.

При каждом выполнении программы, новый отчет добавляется к концу предыдущего отчета. Чтобы очистить содержимое всех отчетов, нужно выполнить команду Очистить отчеты.

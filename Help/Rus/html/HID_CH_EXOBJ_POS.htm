Позиционирование

Перенос, поворот и масштабирование указанной модели STL. Эти команды задают новое положение модели STL относительно системы координат цеха. При нажатии на команду вызывается диалог Позиционирование.


Базовая точка

Характерная точка на габаритном параллелепипеде модели STL. Высвечивается на экране в виде уменьшенного цветного значка системы координат. Задается комбинацией флажков (Min, Middle, Max). По умолчанию минимальные значения X, Y, Z (Min X, Min Y, Min Z). В числовых полях справа от флажков Базовой точки отображаются координаты этой точки в системе координат цеха. Эти числовые поля активны, если выбрана опция Перемещение абсолютное, Поворот, Масштабирование.

Опция Перемещение абсолютное

Служит для перемещения модели в указанные координаты. Например, если в поля XYZ (на картинке выше это значения X-100, Y-100, Z-150), поставить значения "0" и нажать на кнопку Применить, то выбранная модель проекта переместится так, что Базовая точка ее габаритного параллелепипеда будет иметь координаты X0Y0Z0 в системе координат цеха.

Опция Смещение относительное

Служит для относительного перемещения модели на указанные велечины. Если выбрана опция Смещение относительное, то в диалоге становятся видны числовые поля для задания смещений вдоль осей X, Y, Z системы координат цеха. Поставьте в эти поля требуемые значения и нажмите на кнопку Применить. Выбранная модель сместится на эти значения. Например на картинке выше, модели сместится на 200 миллиметров вдоль оси Y. Эта опция включена по умолчанию.

Опция Поворот

Служит для поворота модели вокруг осей XYZ системы координат цеха. В числовых полях Базовой точки, задаются координаты центра поворота. По умолчанию это Базовая точка модели. Углы поворота, это величина поворота вокруг координатных осей. Порядок поворотов XYZ. Поставьте в эти поля требуемые значения и нажмите на кнопку Применить. Выбранная модель проекта будет повернута на эти значения. Например на картинке выше, модель повернется сначало на 45 градусов вокруг оси X, затем на 90 градусов вокруг оси Z. центр поворота X0, Y0, Z0.

Опция Масштабирование

Служит для масштабирование модели относительно указанной точки в системе координат цеха с указанными масштабными коэффициентами по каждой из координатных осей. По умолчанию центр масштабирования это Базовая точка модели. Поставьте требуемые значения для задания центра масштабирования в поля Базовой точки и нужные масштабные коэффициенты в поля Масштаб и нажмите на кнопку Применить. Выбранная модель будет смаштабированна. Например на картинке выше, модели будет вытяута в два раза вдоль оси Z.

Дюйм - ММ

Команда служит для заполнения полей Масштаб коэффициентам, для перевода модели с дюймовыми размерами в модель с миллиметровыми размерами. Нажмите на кнопку Дюйм - ММ, затем на кнопку Применить. Все размеры модели увеличатся в 25.4 раза.

ММ в Дюймы

Команда служит для заполнения полей Масштаб коэффициентам, для перевода модели с миллиметровыми размерами в модель с дюймовыми размерами. Нажмите на кнопку ММ - Дюйм, затем на кнопку Применить. Все размеры модели уменшаться в 0.0393700787401575 раза.

Отменить/Вернуть

Отказаться от последнего преобразования. Шаги не ограничены.

Применить

Выполнить преобразование и остаться в диалоге.

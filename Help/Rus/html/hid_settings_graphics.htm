Настройки графики

Данный диалог позволяет настроить параметры, с которыми будет производиться отображение в Графическом окне.

Изменяемые параметры:

Сетка на инструменте – вы сможете установить следующие параметры сетки инструмента Густая, Средняя, Редкая. Выбирая разные значения этого параметра, Вы будете устанавливать разную частоту сетки, с которой будут отображаться инструменты. Густая сетка позволяет отображать инструмент с максимальным качеством изображения при этом операции с инструментом (различные повороты, перемещение, масштабирование) будут осуществляться несколько медленнее, чем для инструмента с менее густой сеткой (значения параметра Средняя, Редкая). Соответственно для Редкой сетки, операции с инструментом будут выполняться максимально быстро, но с некоторыми потерями качества изображения. Мы рекомендуем работать со Средней сеткой инструмента, при этом будет соблюдаться приемлемая скорость выполнения операций, при достаточно качественном отображении инструмента.

Точность аппроксимации окружности – данный параметр устанавливает максимальное отклонение в пикселях аппроксимирующего окружность многоугольника от аналитической окружности.

Соответственно, чем меньше точность, тем "грубее" (меньшее число сторон многоугольника, соответственно больше ступенчатость окружности) будут отображаться окружности, задающие траектории. Подчёркиваем, этот параметр влияет только на качество отображения окружностей, а не на точность представления окружностей.

Пересчет окружностей происходит при нажатии Вид-> Регенерировать.

Область выбора – размер квадрата в пикселях, внутри которого осуществляется выбор элементов.

Маркер инструмента – данным параметром задаётся размер маркера, маленького прямоугольника, который будет обозначать позицию инструмента при отключённом режиме видимости инструмента.

Маркер привязок – данным параметром также задаёт размер маркера, маленького прямоугольника, который будет отображать привязку к данному кадру программы. Данный маркер появляется при передвижении курсора мыши по траектории движения инструмента с одновременным удерживанием клавиши Ctrl.

Размер осей координат 1 - размеры значка в нижнем левом углу графических окон и ось Z систем координат .
Размер осей координат 2 – размер оси X систем координат
Размер осей координат 3 – размер оси Y систем координат

Коэффициент увеличения лупы –значение, устанавливающее величину коэффициента увеличения лупы. Установка этого параметра вызвана тем, что лупа может увеличивать только один раз и на определённый коэффициент. Значение этого коэффициента не фиксировано, и его значение можно задавать в данном поле.

Точность модели - Задается точность с которой ведется построение модели обработанной заготовки.

Шаг по времени (mc) – временная задержка при пошаговом выполнении программы.

Скорость смены вида – скорость, с какой текущий вид станет другим при выполнении команд смены вида.

Точность экспорта контура – точность для токарной заготовки заданной контуром

Многоцветность заготовки - настроить включение режима раскраски заготовки во время обработки. Если параметр включен и сохранен в файле настройки, то в каждом новом проекте, режим многоцветность заготовки, будет включен. В противном случае, режим многоцветность заготовки будет выключен.

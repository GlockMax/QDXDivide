Диалоговое окно создания инструмента АРТ 7

Фрезерный инструмент может быть задан с помощью параметров стандарта APT 7.

Параметры стандарта APT7:

1. Диаметр инструмента D.

2. Радиус скругления инструмента R.

3. Длина Е. Расстояние от оси инструмента до сентра скругления угла измеренное вдоль оси X.

4: Длина F. Расстояние вдолб оси инструмента до сентра скругления угла.

5: Угол при основании инструмента b.

6: Угол на боковой стороне инструмента a.

7: Высота инструмента H.

Через стандарт APT7, можно задать большинство инструментов. Такой тип инструмента, в основном предназначен для задания концевого инструмента и сверел, с помощью внешнего файла специального формата JOB. JOB файл можно использовать для передачи данных о всех параметрах проекта, из любого внешнего приложения.

Загрузить из NCM …

Команда служит для загрузки в текущий проект, указанных инструментов из другого проекта NCManager. Действия:

1. В текущем проекте из контекстного меню выполните команду ”Загрузить из NCM...” и выберите проект из которого нужно взять инструменты, в появившемся диалоге открытия файлов.

2. Вместо текущего проекта загрузится выбранный. В средине экрана появится диалог «Импорт инструментов», с подсказкой и кнопками выхода из режима.

Не закрывайте этот диалог, пока не отметите нужные инструменты в панели Обекты.

3. Раскройте список инструментов и отметьте галочками те инструменты, которые должны будут использоваться в текущем проекте.

Вы можете зайти в редактирование инструмента, чтобы посмотреть его параметры.

4. Нажмите «OK» для загрузки выбранных инструментов в текущий проект и выхода из режима.

Нажмите «Cancel», для выхода из режима без загрузки инструментов.

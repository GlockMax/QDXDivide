Команда Сохранить (Меню Файл)

Данная команда меню используется для сохранения активного проекта NCManager с использованием его текущего имени и каталога. При сохранении документа без имени, NCManager выводит диалоговое окно сохранения проекта, позволяющее задать имя файла сохраняемого проекта.

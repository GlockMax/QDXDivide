Подвести инструмент

Поставить инструмент в требуемое положение. Вызывается диалог режима «Ручной».

Поставьте с помощью его команд инструмент в положение, которое будет нулем системы координат. Затем выполните команду главного меню Установка Записать ноль программы, чтобы записать это положение в какую-нибудь систему координат (G54-G59)

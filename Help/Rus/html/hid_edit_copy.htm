Команда Копировать (Меню Правка)

Данная команда является одной из команд редактирования в текстовом окне.

Используйте эту команду для того, чтобы скопировать текущий выбранный фрагмент текста в буфер обмена. Эта команда недоступна, если в текстовом окне в настоящий момент не выбрано никаких данных.

При выполнении этой команды данные, предварительно сохранённые в буфере обмена, будут заменены.

Другие способы вызова:

Комбинация клавиш: CTRL + C

или

Контекстное меню текстового редактора

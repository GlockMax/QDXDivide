Команда Вставить (Меню Правка)

Данная команда является одной из команд редактирования в текстовом окне.

Используйте эту команду, чтобы вставить скопированные командой Вырезать или командой Копировать данные из буфера обмена.

Также можно перемещать выделенный фрагмент текста с помощью курсора мыши. Если при перемещении нажать клавишу "Ctrl", то фрагмент текста будет скопирован в новое место.

Другие способы вызова:

Комбинация клавиш: CTRL + V

или

Контекстное меню текстового редактора

Задать тип, размеры и расположение токарной пластины.

Произвольная пластина.

Минимально у этой пластины задается тип (наружный, внутренний, и т.д.) и два угла. Угол с осью X и угол в плане. Точка настройки кромки пластины определится автоматически (мнимая точка). У пластины две режущие кромки. Они выделена цветом Рабочей части.

Имя – Имя пластины. Это имя будет именем всего инструмента, если не задано имя для инструмента в целом и будет именем инструмента в дереве Инструметов.

Тип - Выбор пластину по способу точения. Разверните список и выберите нужный тип из:

Проходной правый.

Расточной правый.

Проходной левый. .

Расточной левый. ..

A - Угол между главной режущей кромкой и осью X.

T - Угол пластины в плане.

L1 – длина главной режущей кромки.

L2 – длина вспомогательной режущей кромки.

R – радиус скругления между режущими кромками.

C– Задний угол пластины

H – Толщина пластины

Коррекция на кромку - заполнить корректоры для учета контурной коррекции на кромку.

Круглая пластина:

Минимально у этой пластины задается радиус. У пластины режущая кромка выделена цветом Рабочей части и точка программирования по умолчанию находится в центре пластине.

Имя – Имя пластины. Это имя будет именем всего инструмента, если не задано имя для инструмента в целом и будет именем инструмента в дереве Инструметов.

R – Радиус пластины

C– Задний угол пластины

H – Толщина пластины

D – Угол ориентации пластины.

Смещение X (DX) Смещение Z (DX) - ввести произвольную величину и знак смещения.

+DX, +DZ, -DX, -DZ - ввести величину и знак смещения. Величина смещения равна радиусу скургления пластины резца.

0 - Обнулить смещение.

Коррекция на кромку - заполнить корректоры для учета контурной коррекции на кромку.

Прорезная пластина

У пластины три режущих кромки вуделенные цветом Рабочей части. Точка программирования совпадает с точкой пересечения левой боковой режущей кромки с средней режущей кромкой.

Имя – Имя пластины. Это имя будет именем всего инструмента, если не задано имя для инструмента в целом и будет именем инструмента в дереве Инструметов.

T – ширина пластины

L – длина пластины

R1 – радиус на левом углу пластины

R2 – радиус на правом углу пластины

A - угол наклона средней режущей кромки.

C– Задний угол пластины

H – Толщина пластины

D – Угол ориентации пластины.

Номер кромки - определить рабочую вершину пластины из двух возможных.

Коррекция на кромку - заполнить корректоры для учета контурной коррекции на кромку.

Смещение X (DX) Смещение Z (DX) - ввести произвольную величину и знак смещения.

+DX, +DZ, -DX, -DZ - ввести величину и знак смещения. Величина смещения равна радиусу скургления пластины резца.

0 - Обнулить смещение.

Пластины по стандарту ISO

У такой пластины две режущие кромки. Исключения составляют круглая пластина. У пластин "Прямоугольная" и "Параллелограмм" три режущих кромки. Точка программирования у всех пластин по умолчанию совпадает с центром радиуса скругления R. Режущие кромки выделены цветом Рабочей части.

Имя – Имя пластины. Это имя будет именем всего инструмента, если не задано имя для инструмента в целом и будет именем инструмента в дереве Инструметов.

Форма – Код формы пластины (квадрат, прямоугольник, ромб, и т.д.). Выберите код из списка.

L1 – длина режущей кромки или радиус для круглой пластины.

L2 – длина второй режущей кромки, если форма пластины не является правильным многоугольником.

R – радиус скругления между режущими кромками. Можно выбрать величину из списка или задать свою.

C– Задний угол пластины

H – Толщина пластины

D – Угол ориентации пластины.

Номер кромки - определить рабочую вершину пластины из двух возможных.

Коррекция на кромку - заполнить корректоры для учета контурной коррекции на кромку.

Смещение X (DX) Смещение Z (DX) - ввести произвольную величину и знак смещения.

+DX, +DZ, -DX, -DZ - ввести величину и знак смещения. Величина смещения равна радиусу скургления пластины резца.

0 - Обнулить смещение.

Параметры резьбовой пластинки

Имя – Имя пластины. Это имя будет именем всего инструмента, если не задано имя для инструмента в целом и будет именем инструмента в дереве Инструметов.

Тип - Выбор пластину по расположению резьбы. Разверните список и выберите нужный тип из:

Наружный правый.

Внутренний правый.

Наружный левый. .

Внутренний левый. ..

A – угол резьбы

T – ширина пластинки.

L – длина пластинки

R – радиус скругления между режущими кромками.

K – расстояние от вершинки пластины до ее левого края.

У такой пластины две режущие кромки и точка программирования зависит от типа обработки и определяется как "точка до касания" с заготовкой.

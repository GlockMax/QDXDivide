Инструменты

Содержит набор инструментов включенных в проект.

При нажатии правой клавиши мыши активизируется контекстное меню, содержащее следующие пункты.

Загрузить - загрузить фрезерный инструмент из библиотеки инструментов.

Создать - новый фрезерный инструмент.

Создать - новый токарный инструмент.

Из проекта NCM ... - Загрузить инструменты из другого проекта. Будет загружен указанный проект, в котором нужно отметить инструменты, которые должны быть загружены в текущий проект.

Когда хотя бы один инструмент задан, то иконка у слова Инструменты изменится и примет вид:

Двойной щелчок левой клавишей мыши или нажатие на крестик раскроет список инструментов заданных в проекте. Подробнее о работе с каждым инструментом читайте в разделе Инструмент.

 

Удалить ненужные - Оставить в списке инструменты, для которых в УП встретился код загрузки (Например, T03M6).

Показать все траектории - Показать траекторию работы всех инструментов во всех программах проекта.

Скрыть все траектории - Скрыть траекторию работы всех инструментов во всех программах проекта.

Инвертировать - Переключить видимость. Т.е. скрыть траекторию для видимых и показать траекторию для скрытых инструментов во всех программах проекта.

Быстрый вызов команды инвертировать:

нажать левой кнопкой мыши на иконку.

 

Свойства - показать панель "Свойства", чтобы задать или посмотреть значения некоторых параметров для инструментов.

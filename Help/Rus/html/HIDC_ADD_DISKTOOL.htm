Создание сложного инструмента

Добавить дисковый инструмент

Команда служит для задания параметров инструментов специальных типов, так называемые дисковые фрезы, размер диаметра которых много больше высоты фрезы.

Для задания инструмента необходимо определить его тип выбором соответствующей иконки и задать необходимые параметры, следуя пояснениям в диалогах.

Приведем примеры специальных инструментов.

Фреза концевая с скруглением на конце.

Фреза с скруглением в произвольном месте.

Дисковые фрезы с фасками;

Дисковые фрезы со скруглениями режущей кромки;



Одно-угловые фрезы;

Двух угловые фрезы;

Инструменты описываемые АРТ 7;

Сложные дисковые фрезы.

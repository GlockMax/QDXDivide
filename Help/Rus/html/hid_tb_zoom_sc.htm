Команда Меньше (Меню Вид)

Данная команда используется для динамического уменьшения или увеличения изображения. Нажмите левую кнопку мыши, и, не отпуская ее перемещайте мышь. Перемещение мыши вниз вызовет уменьшение изображения, вверх - увеличение.

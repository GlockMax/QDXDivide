Имеется два типа основных графических окон. Окно Заготовка и Станок. На экран может быть только одно основное графическое окно. Переключение через главное меню Окно.

Новые панели для отображения в них соответствующей информации во время работы в NCManager. Все панели могут быть размещены в любом месте экрана.

Панель Редактор - служит для просмотра и редактирования текста управляющих программ.

Панель Параметры - служит для просмотра параметров стоики ЧПУ в процессе работы.

Панель Геометрия - служит для просмотра геометрических параметров траектории.

Панель Измерения - служит для просмотра геометрических параметров заготовки.

Панель Регистры - служит для просмотра значений регистров встреченных в тексте программ.

Панель Объекты - служит для управления основными элементами проекта.

Панель Заготовка - служит для отображение процесса имитации обработки.

Панель Станок - служит для отображение процесса имитации обработки.

Панель Ошибки - служит для отображение кадров, в которых найдены столкновения рабочих органов станка.

Изменения в управлении имитацией обработки.

Существует два режима имитации:

Резать - включить режим имитации с удалением материала.

Не резать - включить режим имитации без удаления материала.

Для каждого режима имитации возможен выбор типа перемещения:

Плавно - инструмент движется плавно по траектории.

С учетом подачи - инструмент движется плавно по траектории. Скорость движения инструмента соответствует скорости заданной в управляющей программе.

Быстро - показывается положение инструмента через определенное время работы.

Результат - программа выполняется до активной точки остановки без имитации движения инструмента.

Новый движок для регулирования скорости имитации.

Отображение и управление траекторией инструмента одинаково во всех режимах имитации.

В режимах имитации Быстро и Плавно, в текстовом окне показывается текущий кадр программы.

Можно настроить скорость имитации для различных режимов и типов обработки.

В режиме плавно можно настроить различную скорость для рабочих и холостых перемещений.

Изменения в Панели Объектов

Новые элементы в панели:

Оснастка - служит для размещения моделей оснастки в формате STL.

Деталь - служит для размещения моделей детали в формате STL.

Можно загрузить несколько STL файлов в качестве детали и оснастки.

Операции с STL моделями - Загрузить, Переместить, Повернуть, Масштабировать, Экспортировать в файл.

Токарная заготовка имеет три режима тонирования. Одноцветность, Многоцветность, Зеркальность.

Новый тип фрезерной заготовки, контур вытянуть. Контур может быть импортирован из DXF файла.

Текст каждой программы в отдельности может быть обновлен (вернуться к оригиналу программы).

Изменения в Панели инструментов

Индикаторы - Красным, зеленым, серым цветом показывает текущее состояние некоторых параметров стойки.

 

Текстовое окно - Удалена из панели инструментов и помещено в Панель Редактор.

 

Внешний вид - Удалена, т.к. устарело. Теперь расположение панелей NCManager задается перетаскиванием панелей курсором мыши.

Новые команды главного меню Анализ

Статистика программ - выдается время обработки и длина пути инструментов.

Столкновения - задаются параметры и включения или выключения контроля столкновений станка.

Внизу экрана есть значок, цвет которого показывает статус контроля столкновений.

Серый - контроль столкновений выключен.

Зеленый - контроль столкновений включен, столкновений не обнаружено.

Красный - контроль столкновений включен, обнаружены столкновения.

Список столкновений заносится в панель Ошибки.

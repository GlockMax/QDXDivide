Раскраска текстового окна (меню Правка)

Данная функция выполняет тематическую раскраску текстового окна.

Возможны следующие режимы раскраски:

По типу движения инструмента

Движение инструмента, следовательно, его раскраска рассматривается как комбинация следующих типов:

Холостой ход;

Движение по Отрезку;

Движение по Дуге;

По периодам работы разных инструментов

В данном режиме участки программы, в которых работают разные инструменты, раскрашиваются в цвета соответствующих инструментов.

По способу коррекции

Раскраска по способу коррекции подразделяется на два типа:

Коррекция на длину;

Раскраска программы в режиме Коррекция на длину зависит от направления движения инструмента, и производится по следующим параметрам:

  • Коррекция на движение инструмента вверх;

  • Коррекция на движение инструмента вниз;

  • Коррекция на движение инструмента вправо;

  • Коррекция на движение инструмента влево;

Контурная коррекция;

Раскраска программы в режиме Контурной коррекции осуществляется в зависимости от включения или отключения Контурной коррекции.

По типу охлаждения

Раскраска программы в данном режиме производиться по следующим параметрам:

Охлаждение отключено;

Охлаждение воздухом;

Охлаждение жидкостью;

По величине подачи

В данном режиме цвет текста программы для участков с различной величиной подачи вычисляется из условия, что Красный цвет соответствует минимальной величине подачи, Синий – максимальной величине, а Зелёный соответствует среднему значению между максимальной и минимальной величиной подачи данной программы.

По циклам работы программы

Раскраска программы в режиме Циклов работы программы производиться по следующим параметрам:

Цикл расточки;

Цикл сверления;

Цикл резьбы;

Не производить раскраску текста

Программа раскрасится по общим правилам определенным в настройках цвета.

Изменение цвета того или иного параметра программы, за исключением указанных выше, производиться выбором команды Настройки, Параметры..., закладка Оформление. В появившемся окне в выпадающем списке Элемент выбрать изменяемый параметр затем, нажав на кнопку Цвет выбрать из появившейся цветовой палитры необходимый цвет.

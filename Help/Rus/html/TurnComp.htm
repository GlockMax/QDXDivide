Коррекция на кромку

Команда вызывает диалог для задания величин смещения, радиуса пластинки и кода направления смещения пластины токарного резца.

Номер - номер корректора

X-Смещение - смещение относительно базовой точки пластины по координате X.

Z-Смещение - смещение относительно базовой точки пластины по координате Z.

Радиус - радиус скругления кромеи пластины

Направление - код для задания напрвления режущей кромки.

Номер - добавить корректор в таблицу. Список номеров корректоров, которые используются в текущем проекте, берется из программ автоматически. Вы можете добавить дополнительные номера корректоров, нажав на кнопку с "+".

Кромка - Направление - поясняющая картинка для правильного задания кода направления. Направление осей на картинке: - синим цветом показана ось +Z, красным ось +X.

Задание многокромочного инструмента

Рассмотрим пример задания многокромочного инструмента

На одной пластине могут находиться несколько точек для которых программируется обработка. В управляющей программе, эти точки преключаются с помощью корректора. Например канавочный резец. Правая кромка программируется как, например, 4-ая позиция 25-ый корректор (Т425). Левая кромка программируется как 4-ая позиция и 36-ий корректор (Т436).

Зададим канавочный резец шириной 3мм для прорезании внешней канавки. Для этого, зайдем в диалог задания отрезной пластины, поставим ширину T=3мм и в качестве базовой точки определим левую кромку, поставив галочку у цифры "1" в номере кромки. Т.к. у пластинки есть R1=0.2, а по умолчанию точка настройки совпадает с центром этого радиуса, то нам нужно сместить базовую точку по осям X и Z в отрицательном направлении на величину радиуса (0,2). Сделаем это нажав последовательно на кнопки "-Z" и "-X".

Зададим правую кромку. Для этого мы должны задать смещение вдоль оси Z относительно определенной только что базовой точки (левая кромка). Это смещение мы задаем с помощью команды "Коррекция на кромку" в корректоре под номером 25. Поставьте величину Z-смещения равную ширине пластины (3мм). Для левой кромки мы используем 36 корректор в программе. В этот корректор ничего писать ненужно, т.к. эта кромка определена как базовая.

Ниже приведен пример программы с использованием заданного нами инструмента.

Программа обработки:

T436 Смена инструмента, правая кромка резца
G0 Z-8. X22. Подход к правой кромке канавки
G1 F.1 X15. Прорезание
G0 X22. Выход из канавки
T425 Переход на левую кромку резца
G0 Z-12. Подход к левой кромке канавки
G1 F.1 X15. Прорезание
T436 Переход на правую кромку резца
G1 F.2 Z-8. Зачистка дна канавки
X22. Выход из канавки

Задание корректоров для G41/G42

Если в программе используется контурная коррекция на кромку, то для правильной имитации, в инструменте нужно задать:

1. Определить точку настройки в диалоге задания инструмента (Смещения X и Z).

2. В соответствующий корректор на кромку ввести значения радиуса и код направления пластины.

Команда Вырезать (Меню Правка)

Данная команда является одной из команд редактирования в текстовом окне.

Используйте эту команду, чтобы удалить текущий выбранный фрагмент текста из документа и поместить его в буфер обмена. Эта команда недоступна, если в текстовом окне в настоящий момент не выбрано никаких данных.

При выполнении этой команды данные, предварительно сохранённые в буфере обмена, будут заменены.

Другие способы вызова:

Комбинация клавиш: CTRL + X

Контекстное меню текстового редактора

Команда Запомнить вид с номером (1, 2, … 5) (Меню Вид)

Данная команда используется для запоминания текущего вида под соответствующим именем 1, 2, 3, 4, 5.

Сначало установите требуемый вид в графическом окне, а затем сохраните этот вид под каким-либо номером, выполнив команду Запомнить.

Если с данным именем уже существует сохранённый вид, то он будет заменён.

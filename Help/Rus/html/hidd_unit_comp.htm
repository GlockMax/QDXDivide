Корректоры

Для изменения значения корректора необходимо указать мышью обозначение этого корректора, например D2 или H3. После этого текущее значение выбранного корректора появится в окне редактирования, где его можно изменить.

Обнулить - команда присвоит всем корректорам значение равное "0"

Удалить - команда удалит все корректоры из списка. При следующем вызове команды Исходное состояние - Корректоры, список корректоров будет состоять только из тех, что используются в управляющих программах.

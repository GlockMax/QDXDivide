Панель "Свойства"

Предназначена для быстрого изменения свойств и просмотра справочной информации у указанного элемента в панели «Объекты».

Стойка ЧПУ - свойства для исходного состояния Стойки ЧПУ.

Станок - свойства для станка вцелом.

Модели станка - свойства для каждой модели рабочего органа станка.

Программы - свойства для всех программ в проекте.

Программа - свойства для отдельной программы.

Подпрограммы - свойства для всех подпрограмм проекта. Нет свойств.

Подпрограмма - свойства для отдельной подпрограммы.

Инструменты - свойства для всех инструментов в проекте.

Инструмент - свойства для отдельного инструмента.

Заготовка - свойства для заготовки.

Детали - свойства для всех деталей проекта.

Деталь свойства для каждой деталт.

Оснастка свойства для всех элементов из которых состоит оснастка в проекте.

Элемент оснастки свойства для каждого элемента оснастки.

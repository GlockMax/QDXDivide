Запуск NCManager

Запустите NCManager.

Если NCManager запущен первый раз, то его вид будет таким:

Окно Заготовка. В этом окне отображается траектория в виде отрезков и дуг. При имитации показывается движение инструмента, относительно заготовки.

Панель Объекты - предназначена для отображения и управления основными элементами проекта.

Панель Параметры - предназначена для отображения текущего состояния стойки ЧПУ.

Панель Редактор - предназначена для отображения и редактирования текста управляюшей программы.

В панели Объектов видно что Стойка и Станок заданы.

По-умолчанию загружается стойка ЧПУ – ISO.

Стойка - определяет необходимую информацию для просмотра работы управляющей программы на конкретном станке с ЧПУ.

По-умолчанию загружается станок – 3x_DEFAULT.

3x_DEFAULT - это кинематическая схема 3-х координатного вертикально-фрезерного станка, без моделей рабочих органов. Стол станка перемещается по осям XY, шпиндель по оси Z.

Следующий шаг – загрузка программы.

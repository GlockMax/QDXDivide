Команда Создать фрезерный(Меню Файл)

Данная команда меню используется для создания нового фрезерного проекта в NCManager. Новый проект будет содержать данные заданные в файле прототипа фрезерного проекта. Файл прототипа фрезерного проекта имеет фиксированное имя (Mill.NCMPrt) и должен находиться в папке заданной в меню Настройки - Параметры - Предпочтения - Прототипы.

Фрезерный проект предназначен для имитации обработки на фрезерном станке.

Создать токарную державку

Задать тип, размеры и расположение токарной державки.

Параметры для прямоугольной державки (Брусок):

 

X – длина державки по оси X

Y - длина державки по оси Y

Z - длина державки по оси Z

D - ориентация державки в плоскости XZ

Смещение X смещение державки вдоль оси X

Смещение Z смещение державки вдоль оси Z

Смещение Y смещение державки вдоль оси Y

Параметры для круглой державки (Cone):

X – длина державки

D1 – диаметр державки слева

D2 – диаметр державки справа

Смещение X смещение державки вдоль оси X

Смещение Z смещение державки вдоль оси Z

Смещение Y смещение державки вдоль оси Y

D - ориентация державки в плоскости XZ

Полный диаметр – Если флаг стоит то державка будет полным цилиндром/конусом.

Половина диаметра – Если флаг стоит то державка будет срезана.

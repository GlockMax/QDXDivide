Файл проекта NCManager

Чтобы не повторять все действия по заданию стойки, инструментов, заготовки и еще других параметров, если позже понадобится изучить работу этой же программы, существует понятие - проекта. Файл проекта имеет расширение NCM и в нем содержится информация о загруженных программах, инструменте, заготовки и другие данные.

Данная команда меню используется для создания нового проекта в приложении NCManager. Существует три типа проекта. Фрезерный, токарный и токарно-фрезерный. Данная команда создаст новый проект того типа, который был активным в момент ее вызова. При создании нового проекта, стойка ЧПУ и станок останутся прежними, а программы, заготовка и деталь будут удалены.

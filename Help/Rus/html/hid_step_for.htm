N кадров

Вызов из меню: Работа – Выполнить – N кадров

Быстрый вызов: иконка или нажатие клавиши F3.

Команда служит для выполнения указанного числа кадров программы от текущего места. Число кадров устанавливается в диалоговом окне около иконки. Если число кадров поставить большим, то прервать выполнение можно командами Стоп и Сброс. Если включен один из режимов твердотельной имитации, то будет получена модель обработанной заготовки.

В этой команде все установленные Точки останова игнорируются. Выполнится только указанное число строк текста программы.
